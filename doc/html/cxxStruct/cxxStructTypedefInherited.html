cxxStructTypedefInherited

Links to a typedef inherited by a struct.

Description

The <cxxStructTypedefInherited> element is a child of the <cxxStructInheritsDetail> element. It provides the name of a typedef inherited by a struct and a link to the typedef.

Example

<cxxStructInherits>
    <cxxStructInheritsDetail>
        <cxxStructTypedefInherited href="astruct.xml#atypedef">AStruct::ATypedef</cxxStructTypedefInherited>
    </cxxStructInheritsDetail>
</cxxStructInherits>
