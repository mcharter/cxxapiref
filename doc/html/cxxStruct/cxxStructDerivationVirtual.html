cxxStructDerivationVirtual

Indicates the struct is using virtual inheritance.

Description

The <cxxStructDerivationVirtual> element is contained within a <cxxStructDerivation> element. The name attribute must be set to virtual and the value attribute must be set to true. The base referred to by the <cxxStructDerivationVirtual> element is indicated by using either a <cxxStructBaseStruct>, <cxxStructBaseStruct> or <cxxStructBaseUnion> element as a sibling of <cxxStructDerivationVirtual>.

Example

<cxxStructDerivations>
	<cxxStructDerivation>
		<cxxStructDerivationVirtual name="virtual" value="true"/>
		<cxxStructDerivationAccessSpecifier value="public"/>
		<cxxStructBaseStruct href="struct_base">Base</cxxStructBaseStruct>
	</cxxStructDerivation>
	...
</cxxStructDerivations>
