cxxStructDeclarationFileLineStart

The <cxxStructDeclarationFileLineStart> element specifies the beginning of the struct declaration.

Description

The <cxxStructDeclarationFileLineStart> element is defined within <cxxStructAPIItemLocation>. It contains the line number that the struct declaration begins on. The element is empty but for the required attributes name and value. The attribute name must be set to "lineNumber". The attribute value is set to the line number where struct declaration begins.

Example

<cxxStructDefinition>
    <cxxStructAccessSpecifier value="public" />
    <cxxStructAPIItemLocation>
        <cxxStructDeclarationFile name="filePath" value="s:/sf/os/commsfw/commsconfig/commsdatabaseshim/commdbshim/SCDB/CDBTABLE.CPP" />
        <cxxStructDeclarationFileLineStart name="lineNumber" value="36" />
        <cxxStructDeclarationFileLineEnd name="lineNumber" value="41" />
    </cxxStructAPIItemLocation>
</cxxStructDefinition>
