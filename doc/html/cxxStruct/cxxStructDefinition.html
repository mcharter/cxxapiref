cxxStructDefinition

Contains child elements giving details of a C++ struct definition.

Description

The <cxxStructDefinition> element is defined within <cxxStructDetail>. The child element <cxxStructAPIItemLocation> is mandatory. The child element <cxxStructAccessSpecifier> can indicate the access modifier applied to the members of the struct.

Example

<cxxStructDefinition>
			<cxxStructAccessSpecifier value="public"/>
			<cxxStructDerivations>
				<cxxStructDerivation>
					<cxxStructDerivationAccessSpecifier value="public"/>
					<cxxStructBaseStruct href="struct_c_base.xml#struct_c_base">CBase</cxxStructBaseStruct>
				</cxxStructDerivation>
			</cxxStructDerivations>
			<cxxStructAPIItemLocation>
				<cxxStructDeclarationFile name="filePath" value="C:/src/include/e32base.h"/>
				<cxxStructDeclarationFileLineStart name="lineNumber" value="2784"/>
				<cxxStructDeclarationFileLineEnd name="lineNumber" value="2854"/>
			</cxxStructAPIItemLocation>
		</cxxStructDefinition>
