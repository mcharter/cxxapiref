cxxStructAPIItemLocation

Contains child elements specifying the location of the struct in the source code.

Description

The <cxxStructAPIItemLocation> element is defined within <cxxStructDefinition>. The <cxxStructAPIItemLocation> element must contain the location of the API declaration using <cxxStructDeclarationFile> and <cxxStructDeclarationFileLine>.

Example

<cxxStructAPIItemLocation>
				<cxxStructDeclarationFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxStructDeclarationFileLineStart name="lineNumber" value="765"/>
				<cxxStructDeclarationFileLineEnd name="lineNumber" value="787"/>
			</cxxStructAPIItemLocation>

Contained by

cxxStructDefinition

Contains

cxxStructDeclarationFile cxxStructDeclarationFileLineEnd cxxStructDeclarationFileLineStart

Content Model
