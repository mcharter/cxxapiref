cxxStruct module

Defines a top-level container for all the elements to describe a C++ struct.

Description

The <cxxStruct> topic type contains elements necessary to specify the name of the struct. To describe the member variables and functions insert <cxxFunction> and <cxxVariable> elements into the cxxStruct module.

The cxxStruct topic contains the following elements:

  • cxxStruct
    Provides the name of a C++ struct and has child elements describing the location of the struct and giving detail on the struct members.
  • cxxStructAbstract
    The <cxxStructAbstract> element can be used to mark the struct members as “abstract”..
  • cxxStructAccessSpecifier
    The <cxxStructAccessSpecifier> element describes the access modifier applied to the members of the struct.
  • cxxStructAPIItemLocation
    Contains child elements specifying the location of the struct in the source code.
  • cxxStructBaseClass
    Specifies the base class from which the current struct derives.
  • cxxStructBaseStruct
    Specifies the struct from which the current struct derives.
  • cxxStructBaseUnion
    Specifies the union from which the current struct derives.
  • cxxStructDeclarationFile
    Specifies the source file where the struct is located.
  • cxxStructDeclarationFileLineEnd
    The <cxxStructDeclarationFileLineEnd> element specifies the end of the struct declaration.
  • cxxStructDeclarationFileLineStart
    The <cxxStructDeclarationFileLineStart> element specifies the beginning of the struct declaration.
  • cxxStructDefinition
    Contains child elements giving details of a C++ struct definition.
  • cxxStructDerivation
    Specifies a single struct derivation within a list of struct derivations.
  • cxxStructDerivationAccessSpecifier
    Indicates the access specifier used when a struct inherits from a base class, struct, or union.
  • cxxStructDerivations
    Specifies a list of derivations for a struct.
  • cxxStructDerivationVirtual
    Indicates the struct is using virtual inheritance.
  • cxxStructDetail
    Container for child elements providing a full description of the C++ struct.
  • cxxStructEnumerationInherited
    Provides the name of an enumeration inherited by a struct.
  • cxxStructEnumeratorInherited
    Provides the name of an enumerator inherited by a struct. An enumerator is one of the named constants introduced by an enumeration type.
  • cxxStructFunctionInherited
    Provides the name of a function inherited by a struct.
  • cxxStructInherits
    Cntains the child element <cxxStructInheritsDetail> which gives details of the members inherited by this struct.
  • cxxStructInheritsDetail
    Provides detailed information about the inherited members of the struct.
  • cxxStructNested
    Contains <cxxStructNestedDetail> which gives details of members nested within this C++ struct
  • cxxStructNestedClass
    Provides the name of a class nested within a struct.
  • cxxStructNestedDetail
    Provides details about the members nested within a C++ struct.
  • cxxStructNestedStruct
    Provides the name of a struct nested in a struct.
  • cxxStructNestedUnion
    Provides the name of a union nested in a struct.
  • cxxStructTemplateParameter
    Contains child elements to give information about a parameterized type used by a struct.
  • cxxStructTemplateParameters
    Contains a single <cxxStructTemplateParameter> child element to describe each template parameter declared by a struct.
  • cxxStructTemplateParameterType
    Identifies the template parameter used by a struct template.
  • cxxStructTypedefInherited
    Links to a typedef inherited by a struct.
  • cxxStructVariableInherited
    Provides the name of a variable inherited by a struct.
