cxxStructDeclarationFileLineEnd

The <cxxStructDeclarationFileLineEnd> element specifies the end of the struct declaration.

Description

The <cxxStructDeclarationFileLineEnd> element is defined within <cxxStructAPIItemLocation>. It contains the line number that the struct declaration ends on. The element is empty but for the required attributes name and value. The attribute name must be set to "lineNumber". The attribute value is set to the line number the struct declaration ends on.

Example

<cxxStructDefinition>
    <cxxStructAccessSpecifier value="public" />
    <cxxStructAPIItemLocation>
         <cxxStructDeclarationFile name="filePath" value="s:/sf/os/commsfw/commsconfig/commsdatabaseshim/commdbshim/SCDB/CDBTABLE.CPP" />
        <cxxStructDeclarationFileLineStart name="lineNumber" value="36" />
        <cxxStructDeclarationFileLineEnd name="lineNumber" value="41" />
    </cxxStructAPIItemLocation>
</cxxStructDefinition>
