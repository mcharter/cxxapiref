cxxStructTemplateParameter

Contains child elements to give information about a parameterized type used by a struct.

Description

One or more <cxxStructTemplateParameter> elements reside within the <cxxStructTemplateParameters> container. The <cxxStructTemplateParameter> element uses the mandatory child element <cxxStructTemplateParameterType> to describe the type of the template parameter. <cxxStructTemplateParameter> may also contain the child element <apiDefNote> to describe the parameter further.
