cxxStructDetail

Container for child elements providing a full description of the C++ struct.

Description

The <cxxStructDetail> element is defined within <cxxStruct>. The <cxxStructDetail> element has child elements such as <example> and <section>. These can be used to explain the use of the members of the struct.

Contained by

cxxStruct
