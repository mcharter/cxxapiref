cxxStructBaseClass

Specifies the base class from which the current struct derives.

Description

The <cxxStructBaseClass> element is used within a <cxxStructDerivation> element to declare inheritance from a base class.

Example

<cxxStructDerivation>
    <cxxStructDerivationAccessSpecifier value="public"/>
    <cxxStructBaseClass href="struct_base">Base</cxxStructBaseClass>
</cxxStructDerivation>
