cxxStructDerivations

Specifies a list of derivations for a struct.

Description

The <cxxStructDerivations> element is used within a <cxxStructDefinition> element to specify any base structs, classes, or unions from which a struct derives. The <cxxStructDerivations> element contain at least one <cxxStructDerivation>child element.

Example

<cxxStructDefinition>
	<cxxStructDerivations>
		<cxxStructDerivation>
			...
		</cxxStructDerivation>
		...
	</cxxStructDerivations>
	<cxxStructAPIItemLocation>
		...
	</cxxStructAPIItemLocation>
<cxxStructDefinition>
