cxxUnionDeclarationFileLineEnd

The <cxxUnionDeclarationFileLineEnd> element specifies the end of the union definition.

Description

The <cxxUnionDeclarationFileLineEnd> element is declared within <cxxUnionAPIItemLocation>. It contains the line number that the union declaration ends on. The element is empty but for the required attributes name and value. The attribute name must be set to "lineNumber". The attribute value is set to the line number the union declaration ends on.

Example

<cxxUnionDeclarationFileLineEnd name="lineNumber" value="70"/>
