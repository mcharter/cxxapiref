cxxUnionNestedStruct

The <cxxUnionNestedStruct> element within <cxxUnionNestedDetail> provides the name of a struct nested in a union.

Description

The text in this element is the fully qualified name of the struct that has been nested in a union. There are no mandatory attributes.

Example

<cxxUnionNested>
    <cxxUnionNestedDetail>
        <cxxUnionNestedStruct href="struct_comms.xml#struct_comms_package">CommsFW::TCFLegacyMessagePacker::TUnion::TLegacyPackage</cxxUnionNestedStruct>
        <cxxUnionNestedStruct href="struct_comms.xml#struct_comms_package">CommsFW::TCFLegacyMessagePacker::TUnion::TPackage</cxxUnionNestedStruct>
    </cxxUnionNestedDetail>
</cxxUnionNested>
