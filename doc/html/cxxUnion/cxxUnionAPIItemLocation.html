cxxUnionAPIItemLocation

Contains child elements specifying the location of the union.

Description

The <cxxUnionAPIItemLocation> element is defined within <cxxUnionDefinition. The <cxxUnionAPIItemLocation> element must contain the location of the API declaration using <cxxUnionDeclarationFile> and <cxxUnionDeclarationFileLine>.

Example

<cxxUnionAPIItemLocation>
				<cxxUnionDeclarationFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxUnionDeclarationFileLineStart name="lineNumber" value="765"/>
				<cxxUnionDeclarationFileLineEnd name="lineNumber" value="787"/>
			</cxxUnionAPIItemLocation>

Contained by

cxxUnionDefinition

Contains

cxxUnionDeclarationFile cxxUnionDeclarationFileLineEnd cxxUnionDeclarationFileLineStart
