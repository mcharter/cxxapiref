cxxUnionDeclarationFile

Specifies the source code file containing the union.

Description

The <cxxUnionDeclarationFile> element is defined within <cxxUnionAPIItemLocation>. The only content of the element <cxxUnionDeclarationFile> is the required attributes name and value. name is always "filePath" and value is the actual file path.

Example

<cxxUnionDeclarationFile name="filePath" value="C:/epoc32/include/elements/cftransportmsg.h"/>

Contained by

cxxUnionAPIItemLocation
