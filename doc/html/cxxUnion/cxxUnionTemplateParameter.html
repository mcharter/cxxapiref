cxxUnionTemplateParameter

C++ unions can be parameterized in the same way as C++ classes. The <cxxUnionTemplateParameter> element has child elements to describe a union template parameter.

Description

One or more <cxxUnionTemplateParameter> elements reside within the <cxxUnionTemplateParameters> container. <cxxUnionTemplateParameter> requires the child element <cxxUnionTemplateParameterType> to describe the type of parameter. <cxxUnionTemplateParameter> may also contain the child element <apiDefNote> to describe the parameter further.
