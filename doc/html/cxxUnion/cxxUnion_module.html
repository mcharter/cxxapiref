cxxUnion module

Defines a top-level container for all the elements necessary to describe a C++ union.

Description

The <cxxUnion> topic type contains elements necessary to show the name of the union, the short description of the union, the union syntax, and the union use description. This topic type module also contains all the elements necessary to describe the component variables.

To describe the members of the union insert cxxFunction, cxxDefine, cxxVariable, cxxEnumeration or cxxTypedef elements into the cxxUnion module.

The cxxUnion topic contains the following elements:

  • cxxUnion
    The <cxxUnion> element provides the name of the C++ union and all the elements necessary to describe the members that are introduced by the union declaration.
  • cxxUnionAccessSpecifier
    The <cxxUnionAccessSpecifier> element describes whether the members of the union are "public", "protected" or "private".
  • cxxUnionAPIItemLocation
    Contains child elements specifying the location of the union.
  • cxxUnionDeclarationFile
    Specifies the source code file containing the union.
  • cxxUnionDeclarationFileLineEnd
    The <cxxUnionDeclarationFileLineEnd> element specifies the end of the union definition.
  • cxxUnionDeclarationFileLineStart
    The <cxxUnionDeclarationFileLineStart> element specifies the beginning of the union definition.
  • cxxUnionDefinition
    Contains child elements giving further information about a C++ union.
  • cxxUnionDetail
    The <cxxUnionDetail> element provides detailed information about the C++ union.
  • cxxUnionNested
    Contains a child element <cxxUnionNestedDetail> which gives details of nested members of this C++ union
  • cxxUnionNestedClass
    The <cxxUnionNestedClass> element within <cxxUnionNestedDetail> provides the name of a class nested within a union.
  • cxxUnionNestedDetail
    The <cxxUnionNestedDetail> element provides details about the nested members of a C++ union.
  • cxxUnionNestedStruct
    The <cxxUnionNestedStruct> element within <cxxUnionNestedDetail> provides the name of a struct nested in a union.
  • cxxUnionNestedUnion
    The <cxxUnionNestedUnion> element within <cxxUnionNestedDetail> provides the name of a union nested in a union.
  • cxxUnionTemplateParameter
    C++ unions can be parameterized in the same way as C++ classes. The <cxxUnionTemplateParameter> element has child elements to describe a union template parameter.
  • cxxUnionTemplateParameters
    The <cxxUnionTemplateParameters> element describes one or more union parameters.
  • cxxUnionTemplateParameterType
    The <cxxUnionTemplateParameterType> element refers to a union template parameter.
