cxxUnionTemplateParameterType

The <cxxUnionTemplateParameterType> element refers to a union template parameter.

Description

The <cxxUnionTemplateParameterType> element is defined within the <cxxUnionTemplateParameter> element. It is used along with the sibling element <apiDefNote> to identify and describe a parameterized type declared by a union.
