cxxUnionTemplateParameters

The <cxxUnionTemplateParameters> element describes one or more union parameters.

Description

The <cxxUnionTemplateParameters> element is declared within <cxxUnionDefinition>The <cxxUnionTemplateParameters> element contains one or more <cxxUnionTemplateParameter> elements. Each <cxxUnionTemplateParameter> element describes a single union template parameter.

Example

<cxxUnionTemplateParameters>
  <cxxUnionTemplateParameter>
    <cxxUnionTemplateParameterType>typename T</cxxUnionTemplateParameterType>
    <apiDefNote>The type of the first function object. </apiDefNote>
  </cxxUnionTemplateParameter>
  <cxxUnionTemplateParameter>
    <cxxUnionTemplateParameterType>typename U</cxxUnionTemplateParameterType>
    <apiDefNote>The type of the second function object. </apiDefNote>
  </cxxUnionTemplateParameter>
</cxxUnionTemplateParameters>
