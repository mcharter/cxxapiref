cxxUnionAccessSpecifier

The <cxxUnionAccessSpecifier> element describes whether the members of the union are "public", "protected" or "private".

Description

The <cxxUnionAccessSpecifier> element does not have any children. <cxxUnionAccessSpecifier> uses the attribute value to describe the level of access a union has. valuecan be set to "public", "protected" or "private"; the default is "public".

Example

<cxxUnionAccessSpecifier value="public"/>
