cxxUnionDefinition

Contains child elements giving further information about a C++ union.

Description

The <cxxUnionDefinition> element is defined within <cxxUnionDetail>. The child element <cxxUnionAPIItemLocation> is mandatory. The <cxxUnionAccessSpecifier> child element can be used to indicate the access modifier applied to the members of the union.

Example

<cxxUnionDefinition>
			<cxxUnionAccessSpecifier value="public"/>
			<cxxUnionAPIItemLocation>
				<cxxUnionDeclarationFile name="filePath" value="C:/src/include/e32base.h"/>
				<cxxUnionDeclarationFileLineStart name="lineNumber" value="2784"/>
				<cxxUnionDeclarationFileLineEnd name="lineNumber" value="2854"/>
			</cxxUnionAPIItemLocation>
		</cxxUnionDefinition>
