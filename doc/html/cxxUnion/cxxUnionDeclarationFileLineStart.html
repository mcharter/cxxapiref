cxxUnionDeclarationFileLineStart

The <cxxUnionDeclarationFileLineStart> element specifies the beginning of the union definition.

Description

The <cxxUnionDeclarationFileLineStart> element is declared within <cxxUnionAPIItemLocation>. It contains the line number that the union declaration begins on. The element is empty but for the required attributes name and value. The attribute name must be set to "lineNumber". The attribute value is set to the line number where union declaration begins.

Example

<cxxUnionDeclarationFileLineStart name="lineNumber" value="58"/>
