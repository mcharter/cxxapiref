cxxFriendReturnDesc

Provides a textual description of the return value of a friend function.

Description

The <cxxFriendReturnDesc> element is an optional child element of the <cxxFriendDefinition> element.

Example

<cxxFriendDefinition>
	<cxxFriendReturnDesc>The index in the buffer or -1 if not found or i is &gt;= the buffer length. </cxxFriendReturnDesc>     
</cxxFriendDefinition>
