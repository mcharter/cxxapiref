cxxFriendVolatile

Used to indicate that a friend function has thevolatile cv-qualifier.

Description

The <cxxFriendVolatile> element is present when the friend function has a volatile cv-qualifier . The two mandatory attributes name and value both have the value “volatile”.

See ISO/IEC 14882:2003(E) 7.1.5.1 The cv-qualifiers [dcl.type.cv]

Example

Given this code:

struct X {
    friend void g() volatile;
};

Function g() would have the cxxFriendVolatile element:

<cxxFriendVolatile />

If the mandatory attributes are present they must be set thus:

<cxxFriendVolatile name="volatile" value="volatile" />
