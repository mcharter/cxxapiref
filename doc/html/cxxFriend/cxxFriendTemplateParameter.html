cxxFriendTemplateParameter

Contains the type of the template parameter and any documentation associated with the parameter.

Description

The <cxxFriendTemplateParameter> element had child elements <cxxFriendTemplateParameterType> and <apiDefNote>. These child elements can be used to name template parameter of a friend function template and to describe how the parameter should be used.

See ISO/IEC 14882:2003(E) 14.1 Template parameters [temp.param]
