cxxFriendParameters

Contains information about the friend function parameter declaration clause.

Description

The <cxxFriendParameters> element contains information about the friend function parameter declaration clause that describes the parameters of the function.

See ISO/IEC 14882:2003(E) 8.3.5 Functions [dcl.fct] and ISO/IEC 14882:2003(E) 11.4 Friends [class.friend]

Example

<cxxFriendParameters>
    <cxxFriendParameter>
        <cxxFriendParameterDeclaredType>
            <apiRelation href="class_class__13969__00.xml#class_class__13969__00">Class_13969_00</apiRelation> *</cxxFriendParameterDeclaredType>
        <cxxFriendParameterDefinitionName>p</cxxFriendParameterDefinitionName>
        <apiDefNote />
    </cxxFriendParameter>
    <cxxFriendParameter>
        <cxxFriendParameterDeclaredType>int</cxxFriendParameterDeclaredType>
        <cxxFriendParameterDefinitionName>i</cxxFriendParameterDefinitionName>
        <apiDefNote />
    </cxxFriendParameter>
</cxxFriendParameters>
