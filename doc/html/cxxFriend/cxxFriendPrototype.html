cxxFriendPrototype

Provides a "visual" representation of the friend function declaration.

Description

The are no semantics applied to the PCDATA content of the <cxxFriendPrototype> element. The content should look as similar as possible to the friend function declaration in source code.

Example

<cxxFriendPrototype>friend void friend_set(Class_13969_00 *, int)</cxxFriendPrototype>
