cxxFriendDestructor

Identifies a friend function as a destructor.

Description

The <cxxFriendDestructor> element is present when the friend function is a destructor. This has two mandatory attributes: name and value. Both attributes default to the value "destructor".

A function that uses the cxxFriendDestructor element shall not use the cxxFriendConstructor element

See ISO/IEC 14882:2003(E) 12.4 Destructors [class.dtor]

Example

<cxxFriendDestructor />
