cxxFriendTemplateParameters

Container for elements to describe each of the template parameters of a function template that is also a friend function.

Description

The <cxxFriendTemplateParameters> element contains child elements that describe the template parameters of a friend function template.

See ISO/IEC 14882:2003(E) 14.1 Template parameters [temp.param]

Example

Given this code:

/**
 @tparam T The type of the first function object.
 @tparam U The type of the second function object.
 @return 1 on success, 0 on failure.
 */
template<typename T, typename U>
int templateFunction(T& x, U& y)
{
}

Then function templateFunction would have the following cxxFriendTemplateParameters element:

<cxxFriendTemplateParameters>
  <cxxFriendTemplateParameter>
    <cxxFriendTemplateParameterType>typename T</cxxFriendTemplateParameterType>
    <apiDefNote>The type of the first function object. </apiDefNote>
  </cxxFriendTemplateParameter>
  <cxxFriendTemplateParameter>
    <cxxFriendTemplateParameterType>typename U</cxxFriendTemplateParameterType>
    <apiDefNote>The type of the second function object. </apiDefNote>
  </cxxFriendTemplateParameter>
</cxxFriendTemplateParameters>
