cxxFriendStorageClassSpecifierExtern

Marks a friend function as having external linkage.

Description

The <cxxFriendStorageClassSpecifierExtern> element is present when the friend function has external linkage. The element has two mandatory attributes: name and value. Both of these attributes have the default value “extern”.

See ISO/IEC 14882:2003(E) 7.1.1 Storage class specifiers [dcl.stc] and 3.5 Program and linkage [basic.link]

Example

Given this code:

extern friend void m();

Then function m would have the cxxFriendStorageClassSpecifierExtern element:

<cxxFriendStorageClassSpecifierExtern/>

If the mandatory attributes are present they must be set thus:

<cxxFriendStorageClassSpecifierExtern name="extern" value="extern"/>
