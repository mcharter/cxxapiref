cxxFriendExceptionSpecification

Contains the exception specification of a C++ friend function.

Description

The <cxxFriendExceptionSpecification> element describes the exception specification of a friend function.

See ISO/IEC 14882:2003(E) 15.4 Exception specifications [except.spec].

Example

<cxxFriendExceptionSpecification> throw (int)</cxxFriendExceptionSpecification>
