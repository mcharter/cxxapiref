cxxFriendPureVirtual

Identifies a friend function as pure virtual.

Description

A friend function that uses the pure-specifier is a pure virtual friend function. The <cxxFriendPureVirtual> element has two mandatory attributes, nameand value Both attributes default to the value "pure virtual".

See: ISO/IEC 14882:2003(E) 10.4 Abstract classes [class.abstract]
