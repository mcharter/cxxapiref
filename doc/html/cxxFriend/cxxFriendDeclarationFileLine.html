cxxFriendDeclarationFileLine

Identifies the line number in the file that contains the friendship declaration.

Description

The <cxxFriendDeclarationFileLine> element identifies the line number in the file that contains the friendship declaration. This is not the declaration of the class or function that is the target of the friendship declaration.

The <cxxFriendDeclarationFileLine> element has two attributes. The name attribute always has the value "lineNumber". The value attribute has the line number where the friend keyword is used.

Note: ISO/IEC 14882:2003(E) 16.4 Line control [cpp.line] means that line numbers start from 1.

Example

<cxxFriendDeclarationFileLine name="lineNumber" value="5"/>
