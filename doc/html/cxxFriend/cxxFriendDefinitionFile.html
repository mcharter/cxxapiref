cxxFriendDefinitionFile

Identifies the path to the file that contains the definition of the function.

Description

This identifies the path to the file that contains the definition of the function. (NOTE: This is the friend declaration not the declaration of the friend itself). This has two attributes; name (always has a value "filePath") and value.

Example

<cxxFriendDefinitionFile name="filePath" value="inc/a.h"/>
