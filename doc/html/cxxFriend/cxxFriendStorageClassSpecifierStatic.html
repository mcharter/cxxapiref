cxxFriendStorageClassSpecifierStatic

Marks a friend function as having static linkage.

Description

The <cxxFriendStorageClassSpecifierStatic> element is present when the friend function has static linkage. The <cxxFriendStorageClassSpecifierStatic> element has two mandatory attributes: name and value Both of these attributes have the default value “static”.

See ISO/IEC 14882:2003(E) 7.1.1 Storage class specifiers [dcl.stc] and 3.5 Program and linkage [basic.link]

Example

Given this code:

static friend char* f();

Then friend function f would have the element:

<cxxFriendStorageClassSpecifierStatic/>

If the mandatory attributes are present they must be set thus:

<cxxFriendStorageClassSpecifierStatic name="static" value="static"/>
