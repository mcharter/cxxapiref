cxxFriendStorageClassSpecifierMutable

Marks a friend functions that have a mutable storage-class-specifier.

Description

The <cxxFriendStorageClassSpecifierMutable> element is present when the friend function is mutable. The element has two mandatory attributes: name and value Both of these attributes have the default value “mutable”.

The mutable specifier can be applied only to names of class data members and cannot be applied to names declared const or static, and cannot be applied to reference members.

See ISO/IEC 14882:2003(E) 7.1.1 Storage class specifiers [dcl.stc]

Example

Given this code:

class X {
mutable friend const int* p;
};

Then function X::p would have the element:

<cxxFriendStorageClassSpecifierMutable/>

If the mandatory attributes are present they must be set thus:

<cxxFriendStorageClassSpecifierMutable name="mutable" value="mutable"/>
