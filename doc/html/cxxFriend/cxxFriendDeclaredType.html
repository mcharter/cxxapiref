cxxFriendDeclaredType

Identifies the return type of the friend function.

Description

This element contains the return type of the friend function as PCDATA. There is no semantic significance attached to the text content.

Example

For example this code:

friend void friend_set(Class_13969_00*, int);

Would have a declared type of:

<cxxFriendDeclaredType>void</cxxFriendDeclaredType>
