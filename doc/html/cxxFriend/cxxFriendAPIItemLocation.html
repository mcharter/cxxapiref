cxxFriendAPIItemLocation

Describes the location of a friend class or function in the C++ source code.

Description

Contains child elements showing the filename and line number of the friend class or function.

Example

				<cxxFriendAPIItemLocation>
					<cxxFriendDeclarationFile name="filePath" value="T:/epoc32/include/defect/file_13969.h" />
					<cxxFriendDeclarationFileLine name="lineNumber" value="14" />
					<cxxFriendDefinitionFile name="filePath" value="T:/sf/defect/13969/src/file_13969.cpp" />
					<cxxFriendDefinitionFileLineStart name="lineNumber" value="4" />
					<cxxFriendDefinitionFileLineEnd name="lineNumber" value="7" />
				</cxxFriendAPIItemLocation>
