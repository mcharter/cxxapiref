cxxFriendNameLookup

Provides a unique identifier of the friend function.

Description

The <cxxFriendNameLookup> element provides a way of uniquely identifying a friend function in a body of DITA XML that describes a body of C++ source code.

The free format text in the PCDATA of this element should:
  • be unique within a set of DITA XML files.
  • be human readable.
  • generally follow the lookup techniques used in the C++ standard (ISO/IEC 14882:2003(E)).
  • use a reversible encoding.
