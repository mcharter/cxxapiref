cxxFriendParameterDeclarationName

Describes the name of the parameter used in the declaration of the friend function.

Description

Together with the <cxxFriendParameterDeclaredType> element this element fully describes a single parameter (or argument) to a friend function.

See ISO/IEC 14882:2003(E) 8.3.5 Functions [dcl.fct] and ISO/IEC 14882:2003(E) 11.4 Friends [class.friend]

Example

<cxxFriendParameterDefinitionName>myParam</cxxFriendParameterDefinitionName>
