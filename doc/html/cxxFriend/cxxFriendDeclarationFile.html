cxxFriendDeclarationFile

Identifies the path to the file that contains the statement to mark a target class or function as a friend.

Description

The <cxxFriendDeclarationFile> element identifies the path to the file that contains the declaration of the class or function as a friend. This is the friend declaration not the declaration of the friend itself. The <cxxFriendDeclarationFile> element has two attributes. The name attribute always has the value "filePath" and the value attribute has the name of the file where the friend declaration is made.

Example

<cxxFriendDeclarationFile name="filePath" value="inc/a.h"/>
