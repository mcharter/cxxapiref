cxxFriend module

Defines a top-level container for all the elements necessary to describe an C++ API friend.

Description

The <cxxFriend> topic type contains elements necessary to show the name of the friend, the short description of the friend, the friend syntax, and the friend use description.

Section 11.4 of the C++ standard (ISO/IEC 14882:2003(E)) describes friends.

  • cxxFriend
    The cxxFriend element is the top level container for information about the use of the C++ keyword friend.
  • cxxFriendAPIItemLocation
    Describes the location of a friend class or function in the C++ source code.
  • cxxFriendConstructor
    Identifies a friend function as a constructor.
  • cxxFriendDeclarationFile
    Identifies the path to the file that contains the statement to mark a target class or function as a friend.
  • cxxFriendDeclarationFileLine
    Identifies the line number in the file that contains the friendship declaration.
  • cxxFriendDeclaredType
    Identifies the return type of the friend function.
  • cxxFriendDefinition
    Describes all of the details of a C++ friend function.
  • cxxFriendDefinitionFile
    Identifies the path to the file that contains the definition of the function.
  • cxxFriendDefinitionFileLineEnd
    Identifies the line number in file that contains the end of the definition of the friend function.
  • cxxFriendDefinitionFileLineStart
    Identifies the line number in file that contains the start of the definition of the friend function.
  • cxxFriendDestructor
    Identifies a friend function as a destructor.
  • cxxFriendDetail
    Container for child elements providing information about a friend function.
  • cxxFriendExceptionSpecification
    Contains the exception specification of a C++ friend function.
  • cxxFriendExplicit
    Marks a friend function (that must always be a constructor) as qualified with the C++ keyword explicit.
  • cxxFriendInline
    Identifies the friend function as marked with the C++ keyword inline.
  • cxxFriendNameLookup
    Provides a unique identifier of the friend function.
  • cxxFriendParameter
    Describes a friend function parameter.
  • cxxFriendParameterDeclarationName
    Describes the name of the parameter used in the declaration of the friend function.
  • cxxFriendParameterDeclaredType
    Describes the type of a friend function parameter.
  • cxxFriendParameterDefaultValue
    Describes a default value for a parameter to a friend function.
  • cxxFriendParameterDefinitionName
    Describes the name of the parameter used in the definition of the friend function.
  • cxxFriendParameters
    Contains information about the friend function parameter declaration clause.
  • cxxFriendPrototype
    Provides a "visual" representation of the friend function declaration.
  • cxxFriendPureVirtual
    Identifies a friend function as pure virtual.
  • cxxFriendReturnDesc
    Provides a textual description of the return value of a friend function.
  • cxxFriendScopedName
    Identifies the enclosing scope of the friend function. This is the namespace/class/struct/union within which the friend declaration is located.
  • cxxFriendStorageClassSpecifierExtern
    Marks a friend function as having external linkage.
  • cxxFriendStorageClassSpecifierMutable
    Marks a friend functions that have a mutable storage-class-specifier.
  • cxxFriendStorageClassSpecifierStatic
    Marks a friend function as having static linkage.
  • cxxFriendTemplateParameter
    Contains the type of the template parameter and any documentation associated with the parameter.
  • cxxFriendTemplateParameters
    Container for elements to describe each of the template parameters of a function template that is also a friend function.
  • cxxFriendTemplateParameterType
    Describes the type of the template parameter belonging to a friend function.
  • cxxFriendVirtual
    Identifies a friend function as virtual.
  • cxxFriendVolatile
    Used to indicate that a friend function has thevolatile cv-qualifier.
