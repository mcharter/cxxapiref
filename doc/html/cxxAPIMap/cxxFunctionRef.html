cxxFunctionRef

This element provides a C++ function topic type as a link in a DITA map.

Description

A <cxxFunctionRef> element can contain <topicref> and <topicmeta> elements, allowing you to express navigation or table-of-contents hierarchies, as well as implying relationships between the containing topicref and its children.

Example

<cxxAPIMap id="" title="AMap">
    <cxxFileRef href="afile.xml" navtitle="AFile.h" >
        <cxxDefineRef href="adefine.xml" navtitle="AClass::aDefine" />    			
        <cxxEnumerationRef href="anenum.xml" navtitle="AClass::anEnum" />
        <cxxFunctionRef href="afunction.xml" navtitle="AClass::aFunction" />
        <cxxTypedefRef href="atypedef.xml" navtitle="AClass::aTypeDef" />
        <cxxVariableRef href="avariable.xml" navtitle="AClass::aVariable" />		
    </cxxFileRef>
</cxxAPIMap>
