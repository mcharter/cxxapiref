cxxClassRef

Provides a C++ class topic type as a link in a DITA map.

Description

A <cxxClassRef> element can contain other <cxxClassRef> elements or <cxxStructRef> and <cxxUnionRef> elements; allowing you to express navigation or table-of-contents hierarchies.

Example

<cxxAPIMap id="amap" title="AMap">
	<cxxFileRef href="afile.xml" navtitle="afile.h" />
	<cxxClassRef href="aclass.xml" navtitle="AClass" />
	<cxxStructRef href="astruct.xml" navtitle="AClass::AStruct" />
	<cxxUnionRef href="aunion.xml" navtitle="AClass::AUnion" />
	<cxxNamespaceRef href="anamespace.xml" navtitle="AClass::ANamespace" />
</cxxAPIMap>
