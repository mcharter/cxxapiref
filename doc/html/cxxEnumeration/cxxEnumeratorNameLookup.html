cxxEnumeratorNameLookup

Provides a unique identifier of the enumerator.

Description

The <cxxEnumeratorNameLookup> element provides a way of uniquely identifying an enumerator in a body of DITA XML that describes a body of C++ source code.

The free format PCDATA text in this element should:
  • be unique within a set of DITA XML files.
  • be human readable.
  • follow the lookup techniques use in the C++ standard (ISO/IEC 14882:2003(E)).
  • use a reversible encoding.
