cxxEnumerationReimplemented

Identifies enumerations that have been re-implemented.

Description

The <cxxEnumerationReimplemented> element identifies and links to enumerations that have been re-implemented.

Example

<cxxEnumerationReimplemented href="class_c_active.xml#class_c_active_1a4bb7613b">CActive::dSpare</cxxEnumerationReimplemented>
