cxxEnumerationPrototype

Provides a "visual" representation of the enumeration declaration.

Description

This element provides a "visual" representation of the enumeration declaration as PCDATA. The are no semantics applied to the content but the intention is that the content should look as similar as possible to the enumeration declaration in source code.
