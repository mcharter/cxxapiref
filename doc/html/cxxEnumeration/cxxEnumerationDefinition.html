cxxEnumerationDefinition

Container for child elements describing the visiblilty and location of a C++ enum..

Description

The <cxxEnumerationDefinition> element is defined within <cxxEnumerationDetail>. The child element <cxxEnumerationAPIItemLocation> is mandatory.

Example

<cxxEnumerationDefinition>
			<cxxEnumerationAccessSpecifier value="public"/>
			<cxxEnumerationAPIItemLocation>
				<cxxEnumerationDeclarationFile name="filePath" value="C:/src/include/e32base.h"/>
				<cxxEnumerationDeclarationFileLineStart name="lineNumber" value="2784"/>
				<cxxEnumerationDeclarationFileLineEnd name="lineNumber" value="2854"/>
			</cxxEnumerationAPIItemLocation>
		</cxxEnumerationDefinition>

Contained by

cxxEnumerationDetail

Contains

cxxEnumerationAPIItemLocation cxxEnumerationAccessSpecifier cxxEnumerationNameLookup cxxEnumerationPrototype cxxEnumerationReimplemented cxxEnumerationScopedName cxxEnumerators
