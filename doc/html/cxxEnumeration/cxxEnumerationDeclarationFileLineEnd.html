cxxEnumerationDeclarationFileLineEnd

The <cxxEnumerationDeclarationFileLineEnd> element specifies the end of the enumeration declaration.

Description

The <cxxEnumerationDeclarationFileLineEnd> element is declared within <cxxEnumerationAPIItemLocation>. It contains the line number that the enumeration declaration ends on. The element is empty but for the required attributes name and value. The attribute name must be set to "lineNumber". The attribute value is set to the line number the enumeration declaration ends on.

Example

<cxxEnumerationAPIItemLocation>
    <cxxEnumerationDeclarationFile name="filePath" value="C:/src/master/sf/mw/animation/src/SpriteAnimationServer.cpp"/>
    <cxxEnumerationDeclarationFileLineStart name="lineNumber" value="1719"/>
    <cxxEnumerationDeclarationFileLineEnd name="lineNumber" value="1727"/>
</cxxEnumerationAPIItemLocation>
