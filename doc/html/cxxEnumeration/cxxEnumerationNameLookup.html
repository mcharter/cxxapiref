cxxEnumerationNameLookup

Provides a unique identifier of the enumeration.

Description

The <cxxEnumerationNameLookup> element provides a way of uniquely identifying a enumeration in a body of DITA XML that describes a body of C++ source code.

The free format PCDATA text should:
  • be unique within a set of DITA XML files.
  • be human readable.
  • generally follow the lookup techniques use in the C++ standard (ISO/IEC 14882:2003(E)).
  • use a reversible encoding.
