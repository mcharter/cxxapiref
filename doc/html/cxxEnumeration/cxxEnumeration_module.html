cxxEnumeration module

Defines a top-level container for all the elements necessary to describe an C++ enumeration.

Description

The <cxxEnumeration> topic type contains elements to show the name and members of an enumeration.

The cxxEnumeration topic contains the following elements:

  • cxxEnumeration
    The <cxxEnumeration> element contains all the elements needed to describe a C++ enumeration.
  • cxxEnumerationAccessSpecifier
    The <cxxEnumerationAccessSpecifier> element describes the access specifier for the enumeration.
  • cxxEnumerationAPIItemLocation
    Describes the location of an enumeration in the C++ source code.
  • cxxEnumerationDeclarationFile
    Specifies the file in which the enum declaration is present.
  • cxxEnumerationDeclarationFileLineEnd
    The <cxxEnumerationDeclarationFileLineEnd> element specifies the end of the enumeration declaration.
  • cxxEnumerationDeclarationFileLineStart
    The <cxxEnumerationDeclarationFileLineStart> element specifies the beginning of the enumeration declaration.
  • cxxEnumerationDefinition
    Container for child elements describing the visiblilty and location of a C++ enum..
  • cxxEnumerationDetail
    Contains child elements to provide detailed information about the C++ enumeration.
  • cxxEnumerationNameLookup
    Provides a unique identifier of the enumeration.
  • cxxEnumerationPrototype
    Provides a "visual" representation of the enumeration declaration.
  • cxxEnumerationReimplemented
    Identifies enumerations that have been re-implemented.
  • cxxEnumerationScopedName
    Specifies the enclosing namespace/class/struct/union in which the union is defined.
  • cxxEnumerator
    The <cxxEnumerator> element within the <cxxEnumerators> element provides the details of an enumerator defined within an enumeration.
  • cxxEnumeratorAPIItemLocation
    Contains child elements to specify the location of the enumeration in the source code.
  • cxxEnumeratorDeclarationFile
    Specifies the file in which the enumeration is declared.
  • cxxEnumeratorDeclarationFileLine
    Specifies the line number on which the enumeration is declared.
  • cxxEnumeratorInitialiser
    Specifies the initial value of the enumerator.
  • cxxEnumeratorNameLookup
    Provides a unique identifier of the enumerator.
  • cxxEnumeratorPrototype
    Provides a "visual" representation of the enumerator declaration.
  • cxxEnumerators
    The <cxxEnumerators> element contains a <cxxEnumerator> element for each enumerator in an enumeration.
  • cxxEnumeratorScopedName
    Specifies the enclosing scope containing the enumerator.
