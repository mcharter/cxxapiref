cxxEnumerationAccessSpecifier

The <cxxEnumerationAccessSpecifier> element describes the access specifier for the enumeration.

Description

The <cxxEnumerationAccessSpecifier> element contains the optional name and value attributes. The name attribute can only be set to "access". The value attribute must be either “public”, “protected” or “private”. The default value of the value attribute is "public".

Example

<cxxEnumerationAccessSpecifier value="public"/>
