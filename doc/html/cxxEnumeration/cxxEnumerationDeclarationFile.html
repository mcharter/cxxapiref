cxxEnumerationDeclarationFile

Specifies the file in which the enum declaration is present.

Description

The <cxxEnumerationDeclarationFile> element is defined within <cxxEnumerationAPIItemLocation>. The only content of the element <cxxEnumerationDeclarationFile> is the required attributes name and value. name is always "filePath" and value is the actual file path.

Example

<cxxEnumerationAPIItemLocation>
    <cxxEnumerationDeclarationFile name="filePath" value="C:/src/include/gdi.h"/>
    <cxxEnumerationDeclarationFileLineStart name="lineNumber" value="1719"/>
    <cxxEnumerationDeclarationFileLineEnd name="lineNumber" value="1727"/>
</cxxEnumerationAPIItemLocation>
