cxxEnumeratorAPIItemLocation

Contains child elements to specify the location of the enumeration in the source code.

Description

The <cxxEnumeratorAPIItemLocation> element is defined within <cxxEnumerator>. The <cxxEnumeratorAPIItemLocation> element must contain the location of the API declaration using the child elements <cxxEnumeratorDeclarationFile> and <cxxEnumeratorDeclarationFileLine>.

Example

<cxxEnumeratorAPIItemLocation>
				<cxxEnumeratorDeclarationFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxEnumeratorDeclarationFileLine name="lineNumber" value="775"/>
			</cxxEnumeratorAPIItemLocation>

Contained by

cxxEnumerator

Contains

cxxEnumeratorDeclarationFile cxxEnumeratorDeclarationFileLine
