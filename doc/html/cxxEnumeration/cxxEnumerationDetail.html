cxxEnumerationDetail

Contains child elements to provide detailed information about the C++ enumeration.

Description

The <cxxEnumerationDetail> element is defined within <cxxEnumeration>. You can use child elements such as <example> and <section> to provide detail about the use of the enumeration.
