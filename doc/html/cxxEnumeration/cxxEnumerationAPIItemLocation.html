cxxEnumerationAPIItemLocation

Describes the location of an enumeration in the C++ source code.

Description

The <cxxEnumerationAPIItemLocation> element must contain the location of the enumeration using the child elements <cxxEnumerationDeclarationFile> and <cxxEnumerationDeclarationFileLine>.

Example

<cxxEnumerationAPIItemLocation>
    <cxxEnumerationDeclarationFile name="filePath" value="C:/src/include/w32adll.h"/>
    <cxxEnumerationDeclarationFileLine name="lineNumber" value="1072"/>
</cxxEnumerationAPIItemLocation>
