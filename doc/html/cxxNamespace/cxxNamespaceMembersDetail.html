cxxNamespaceMembersDetail

<cxxNamespaceMembersDetail> contains links to topics that describe members of a C++ namespace

Description

The <cxxNamespaceMembersDetail> element is a child of the <cxxNamespaceMembers> element. It contains links to topics for members defined in a C++ namespace.
