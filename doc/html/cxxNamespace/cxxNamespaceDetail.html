cxxNamespaceDetail

Contains child elements specifying the file or files where a C++ namespace is used.

Description

A C++ namespace can be used to qualify the names of classes, functions and variables. Use of a namespace can span multiple source files. The <cxxNamespaceDetail> element contains child elements providing this detailed information about the use of a C++ namespace.

Example

<cxxNamespaceDetail>
		<cxxNamespaceDefinition>
			<cxxNamespaceAPIItemLocation>
				<cxxNamespaceDeclarationFile name="filePath" value="t:/epoc32/include/feature/file_14749.h" />
				<cxxNamespaceDeclarationFile name="filePath" value="t:/epoc32/include/feature/file_14749_func.h" />
				<cxxNamespaceDeclarationFile name="filePath" value="t:/sf/feature/14749/src/file_14749.cpp" />
				<cxxNamespaceDeclarationFile name="filePath" value="t:/sf/feature/14749/src/file_14749_func.cpp" />
			</cxxNamespaceAPIItemLocation>
		</cxxNamespaceDefinition>
		<apiDesc>
			<p otherprops="linkagainst">14749.lib</p><p>Example adapted from ISO/IEC 14882:2003(E) 7.3.1 Namespace definition [namespace.def]</p>
			<p>Even more documentation for namespace Outer from <xref format="html" href="file:">file:</xref> <xref href="file__14749_8h.xml#file__14749_8h_1a80752018587ae8dc67f2b250db6e7eb8">file_14749_func.h</xref></p>
			<p>More documentation for namespace Outer from <xref format="html" href="file:">file:</xref> <xref href="file__14749_8h.xml#file__14749_8h_1a80752018587ae8dc67f2b250db6e7eb8">file_14749_class.h</xref> </p>
		</apiDesc>
	</cxxNamespaceDetail>
	<cxxNamespaceMembers>
