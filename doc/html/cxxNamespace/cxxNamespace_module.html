cxxNamespace module

Defines a top-level container for all the elements necessary to describe a C++ namespace.

Description

The <cxxNamespace> topic type contains elements to specify the name of the namespace and to describe the variables and functions within the namespace.

The cxxNamespace topic contains the following elements:

  • cxxNamespace
    cxxNamespace is a topic type that contains all the elements to describe a c++ namespace
  • cxxNamespaceDetail
    Contains child elements specifying the file or files where a C++ namespace is used.
  • cxxNamespaceDefinition
    Has child elements providing further information about a C++ namespace.
  • cxxNamespaceAPIItemLocation
    Specifies the location of the C++ namespace.
  • cxxNamespaceDeclarationFile
    Specifies the file in which the C++ namespace is located.
  • cxxNamespaceMembers
    <cxxNamespaceMembers> is a container element that can provide links to the C++ identifiers defined within the namespace.
  • cxxNamespaceMembersDetail
    <cxxNamespaceMembersDetail> contains links to topics that describe members of a C++ namespace
  • cxxNamespaceMemberClass
    <cxxNamespaceMemberClass> is a link to a cxxClass topic. The cxxClass topic that is linked to describes a class defined within a C++ namespace.
  • cxxNamespaceMemberStruct
    <cxxNamespaceMemberStruct> is a link to a cxxStruct topic. The cxxStruct topic that is linked to describes a struct defined within the namespace.
  • cxxNamespaceMemberUnion
    <cxxNamespaceMemberUnion> is a link to a cxxUnion topic. The cxxUnion topic that is linked to describes a union defined within the namespace.
  • cxxNamespaceMemberNamespace
    <cxxNamespaceMemberNamespace> is a link to a cxxNamespace topic. The cxxNamespace topic that is linked to describes a namespace nested within another namespace.
