cxxNamespaceAPIItemLocation

Specifies the location of the C++ namespace.

Description

The <cxxNamespaceAPIItemLocation> element is defined within <cxxNamespaceDefinition>. The <cxxNamespaceAPIItemLocation> element must contain the location of the namespace using <cxxNamespaceDeclarationFile>.

Example

<cxxNamespaceDefinition>
			<cxxNamespaceAPIItemLocation>
				<cxxNamespaceDeclarationFile name="filePath" value="t:/epoc32/include/feature/file_14749.h" />
				<cxxNamespaceDeclarationFile name="filePath" value="t:/epoc32/include/feature/file_14749_func.h" />
				<cxxNamespaceDeclarationFile name="filePath" value="t:/sf/feature/14749/src/file_14749.cpp" />
				<cxxNamespaceDeclarationFile name="filePath" value="t:/sf/feature/14749/src/file_14749_func.cpp" />
			</cxxNamespaceAPIItemLocation>
		</cxxNamespaceDefinition>
