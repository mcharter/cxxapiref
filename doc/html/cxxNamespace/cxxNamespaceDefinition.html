cxxNamespaceDefinition

Has child elements providing further information about a C++ namespace.

Description

The <cxxNamespaceDefinition> element requires the child element <cxxNamespaceAPIItemLocation>.

Example

<cxxNamespaceDefinition>
			<cxxNamespaceAccessSpecifier value="public"/>
			<cxxNamespaceDerivations>
				<cxxNamespaceDerivation>
					<cxxNamespaceDerivationAccessSpecifier value="public"/>
					<cxxNamespaceBaseClass href="namespace.xml#namespace">CBase</cxxNamespaceBaseClass>
				</cxxNamespaceDerivation>
			</cxxNamespaceDerivations>
			<cxxNamespaceAPIItemLocation>
				<cxxNamespaceDeclarationFile name="filePath" value="C:/src/include/e32base.h"/>
				<cxxNamespaceDeclarationFileLine name="lineNumber" value="2812"/>
			</cxxNamespaceAPIItemLocation>
		</cxxNamespaceDefinition>
