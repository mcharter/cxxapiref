cxxNamespaceMemberClass

<cxxNamespaceMemberClass> is a link to a cxxClass topic. The cxxClass topic that is linked to describes a class defined within a C++ namespace.

Description

The <cxxNamespaceMemberClass> element is defined inside a <cxxNamespaceMembersDetail> element. The text inside the <cxxNamespaceMemberClass> element can be used to give the name of the class that is being linked to. An href or keyref attribute is used to provide a reference to the cxxClass topic.

Example

<cxxNamespaceMembers>
    <cxxNamespaceMembersDetail>
        <cxxNamespaceMemberClass href="class_all_types_of_members_1_1_member_class.xml#class_all_types_of_members_1_1_member_class">AllTypesOfMembers::MemberClass</cxxNamespaceMemberClass>
        <cxxNamespaceMemberStruct href="struct_all_types_of_members_1_1_member_struct.xml#struct_all_types_of_members_1_1_member_struct">AllTypesOfMembers::MemberStruct</cxxNamespaceMemberStruct>
        <cxxNamespaceMemberClass href="union_all_types_of_members_1_1_member_union.xml#union_all_types_of_members_1_1_member_union">AllTypesOfMembers::MemberUnion</cxxNamespaceMemberClass>
        <cxxNamespaceMemberNamespace href="namespace_all_types_of_members_1_1_member_namespace.xml#namespace_all_types_of_members_1_1_member_namespace">AllTypesOfMembers::MemberNamespace</cxxNamespaceMemberNamespace>
    </cxxNamespaceMembersDetail>
</cxxNamespaceMembers>
