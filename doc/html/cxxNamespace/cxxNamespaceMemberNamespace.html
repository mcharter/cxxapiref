cxxNamespaceMemberNamespace

<cxxNamespaceMemberNamespace> is a link to a cxxNamespace topic. The cxxNamespace topic that is linked to describes a namespace nested within another namespace.

Description

C++ namespaces can be nested. The <cxxNamespaceMemberNamespace> element is defined inside a <cxxNamespaceMembersDetail> element. The text inside the <cxxNamespaceMemberNamespace> element can be used to give the name of the namespace that is being linked to. An href or keyref attribute is used to provide a reference to the cxxNamespace topic.

Example

<cxxNamespaceMembers>
    <cxxNamespaceMembersDetail>
        <cxxNamespaceMemberClass href="class_all_types_of_members_1_1_member_class.xml#class_all_types_of_members_1_1_member_class">AllTypesOfMembers::MemberClass</cxxNamespaceMemberClass>
        <cxxNamespaceMemberStruct href="struct_all_types_of_members_1_1_member_struct.xml#struct_all_types_of_members_1_1_member_struct">AllTypesOfMembers::MemberStruct</cxxNamespaceMemberStruct>
        <cxxNamespaceMemberUnion href="union_all_types_of_members_1_1_member_union.xml#union_all_types_of_members_1_1_member_union">AllTypesOfMembers::MemberUnion</cxxNamespaceMemberUnion>
        <cxxNamespaceMemberNamespace href="namespace_all_types_of_members_1_1_member_namespace.xml#namespace_all_types_of_members_1_1_member_namespace">AllTypesOfMembers::MemberNamespace</cxxNamespaceMemberNamespace>
    </cxxNamespaceMembersDetail>
</cxxNamespaceMembers>
