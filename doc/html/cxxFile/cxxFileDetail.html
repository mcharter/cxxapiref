cxxFileDetail

Provides information about the properties of a file.

Description

The <cxxFileDetail> element provides information about the file containing the global API. Child elements describe the name, location and usage of the file.

Example

<cxxFile id="afile">
	<apiName>afile.h</apiName>
	<cxxFileDetail>
		<cxxFileAPIItemLocation>
			<cxxFileDeclarationFile name="filePath" value="C:/src/afile.h"/>
		</cxxFileAPIItemLocation>
		<apiDesc>afile.h demonstrates the simple use cxxFile and related elements.</apiDesc>
	</cxxFileDetail>
</cxxFile>
