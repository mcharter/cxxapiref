cxxFile

The cxxFile element contains information about items declared in a file in C++ (for example a header file).

Description

The <cxxFile> element contains information about C++ items defined in a header file e.g. enums or typdefs

It can also contain links to other topics for items e.g. classes that have been declared in the header file

The <cxxFile> element requires an id attribute.

The <cxxFile> element must contain the file name within the element <apiname>. <cxxFile> can also contain zero or more of the following elements to describe the global information:

  • <cxxFunction>
  • <cxxDefine>
  • <cxxVariable>
  • <cxxEnumeration>
  • <cxxTypedef>

Example

<cxxFile id="afile">
	<apiName>afile.h</apiName>
	<cxxFileDetail>
		<cxxFileAPIItemLocation>
			<cxxFileDeclarationFile name="filePath" value="C:/src/afile.h"/>
		</cxxFileAPIItemLocation>
		<apiDesc>afile.h demonstrates the simple use cxxFile and related elements.</apiDesc>
	</cxxFileDetail>
	<cxxVariable id="avar">
		<apiName>AVar</apiName>
		<cxxVariableDetail>
			...
		</cxxVariableDetail>
	</cxxVariable>
</cxxFile>
