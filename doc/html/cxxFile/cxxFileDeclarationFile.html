cxxFileDeclarationFile

Defines the location of the file containing the global API.

Description

The <cxxFileDeclarationFile> element is a child of the <cxxFileAPIItemLocation> element. The location of the file is described using the mandatory attributes name and value. name contains the default value "filepath" and value contains the full path to the file.

Example

	<cxxFileAPIItemLocation>
		<cxxFileDeclarationFile name="filePath" value="C:/src/include/cfshared.h" />
	</cxxFileAPIItemLocation>
