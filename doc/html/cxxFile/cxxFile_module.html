cxxFile module

The cxxFile module defines a top-level container for all the elements necessary to describe an API declared at the global scope.

Description

The <cxxFile> topic type contains elements necessary to show the name of the file, and also the global variables, functions, preprocessor directives,, enumerations and typedefs contained within.

The cxxFile topic contains the following elements:

  • cxxFile
    The cxxFile element contains information about items declared in a file in C++ (for example a header file).
  • cxxFileAPIItemLocation
    A container for the element that describes the location of the file containing the global APIs.
  • cxxFileDeclarationFile
    Defines the location of the file containing the global API.
  • cxxFileDetail
    Provides information about the properties of a file.
