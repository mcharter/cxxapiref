cxxDefineParameters

A #define that takes arguments (or parameters) is commonly called a macro. This element allows you to describe the purpose of the identifier list that makes up the macro arguments.

Description

When a preprocessor #define directive is used as a macro, the <cxxDefineParameters> element contains child elements that describe the macro parameters.

Example

<cxxDefineParameters>
    <cxxDefineParameter>
        <cxxDefineParameterDeclarationName>prmIndex</cxxDefineParameterDeclarationName>
        <apiDefNote/>
    </cxxDefineParameter>
</cxxDefineParameters>
