cxxDefineNameLookup

Provides a unique identifier of a preprocessor #define directive.

Description

The <cxxDefineNameLookup> element provides a way of uniquely identifying a #define within a body of C++ source code.

The PCDATA of this element is free format text. The text should:
  • be unique within a set of DITA XML files.
  • be human readable.
  • follow the lookup techniques use in the C++ standard (ISO/IEC 14882:2003(E)).
  • use a reversible encoding.

Example

<cxxDefineNameLookup>ENABLE_NETWORKING_PERFMETRICS</cxxDefineNameLookup>
