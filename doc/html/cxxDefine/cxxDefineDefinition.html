cxxDefineDefinition

Contains child elements that provide a description of a preprocessor definition introduced through the #define directive.

Description

The <cxxDefineDefinition> element is defined within <cxxDefineDetail>. It provides information about a preprocessor directive, such as the file where it is located, the fact that it is globally accessible (i.e. â€œpublic), and the name of preprocessor symbol that has been introduced.

Example

<cxxDefineDefinition>
    <cxxDefinePrototype>#define ENABLE_NETWORKING_PERFMETRICS</cxxDefinePrototype>
    <cxxDefineNameLookup>ENABLE_NETWORKING_PERFMETRICS</cxxDefineNameLookup>
    <cxxDefineParameters/>
    <cxxDefineAPIItemLocation>
        <cxxDefineDeclarationFile name="filePath" value="C:/EPOC/master/sf/os/commsfw/src/cfperfmetrics.cpp"/>
        <cxxDefineDeclarationFileLine name="lineNumber" value="17"/>
    </cxxDefineAPIItemLocation>
</cxxDefineDefinition>
