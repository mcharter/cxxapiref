cxxDefineDeclarationFileLine

Specifies the line number containing the #define.

Description

The <cxxDefineDeclarationFileLine> element is defined within <cxxDefineAPIItemLocation>. The element <cxxDefineDeclarationFileLine>has no children. It has the required attributes name and value. name is always set to "lineNumber" and value is set to the line number that the #define appears on.

Example

<cxxDefineAPIItemLocation>
    <cxxDefineDeclarationFile name="filePath" value="C:/src/SCOMM/cs_roles.cpp"/>
    <cxxDefineDeclarationFileLine name="lineNumber" value="35"/>
</cxxDefineAPIItemLocation>
