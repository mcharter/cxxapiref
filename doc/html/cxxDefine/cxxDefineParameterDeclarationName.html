cxxDefineParameterDeclarationName

The <cxxDefineParameterDeclarationName> element specifies the name of a parameter used by a preprocessor macro.

Description

A #define can take one or more arguments (or parameters). Such a #define is usually known as a macro. The value of the parameters are supplied by the code that uses the macro. Use the <cxxDefineParameterDeclarationName> element together with the <apiDefNote> element to give information about the intended use of macro parameters.

<cxxDefineParameterDeclarationName> has no child elements.

Example

<cxxDefineDefinition>
    <cxxDefinePrototype>#define MSG_PRM(prmIndex)</cxxDefinePrototype>
    <cxxDefineNameLookup>MSG_PRM</cxxDefineNameLookup>
    <cxxDefineParameters>
        <cxxDefineParameter>
            <cxxDefineParameterDeclarationName>prmIndex</cxxDefineParameterDeclarationName>
            <apiDefNote/>
        </cxxDefineParameter>
    </cxxDefineParameters>
    ...
</cxxDefineDefinition>
