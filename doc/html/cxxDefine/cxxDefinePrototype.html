cxxDefinePrototype

Gives a view of a preprocessor #define directive as it looks in the source code.

Description

The <cxxDefinePrototype> element provides a "visual" representation of a preprocessor #define . The content is PCDATA. It should look as similar as possible to the #define in the source code. No semantics are applied to the content.

Example

<cxxDefinePrototype>#define ENABLE_NETWORKING_PERFMETRICS</cxxDefinePrototype>
