cxxDefineAPIItemLocation

Specifies the location of a preprocessor symbol introduced using #define.

Description

The <cxxDefineAPIItemLocation> element is defined within <cxxDefineDefinition>. The <cxxDefineAPIItemLocation> element must contain the child elements <cxxDefineDeclarationFile> and <cxxDefineDeclarationFileLine>. These elements indicate the location of the #define.

Example

<cxxDefineAPIItemLocation>
    <cxxDefineDeclarationFile name="filePath" value="C:/EPOC/master/sf/os/commsfw/commsfwsupport/commselements/commsfw/src/cfperfmetrics.cpp"/>
    <cxxDefineDeclarationFileLine name="lineNumber" value="17"/>
</cxxDefineAPIItemLocation>
