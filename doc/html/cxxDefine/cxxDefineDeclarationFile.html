cxxDefineDeclarationFile

Specifies the file containing the #define.

Description

The <cxxDefineDeclarationFile> element is a child of <cxxDefineAPIItemLocation>. The element <cxxDefineDeclarationFile> does not have any children. It has the required attributes name and value. name is always set to "filePath" and value is set to the file containing the #define.

Example

<cxxDefineDeclarationFile name="filePath" value="C:/EPOC/master/sf/os/commsfw/commsfw/src/cfperfmetrics.cpp"/>

Contained by

cxxDefineAPIItemLocation
