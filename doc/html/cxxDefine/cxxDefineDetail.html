cxxDefineDetail

The <cxxDefineDetail> element contain child elements to fully describe a #define preprocessor directive.

Description

The <cxxDefineDetail> element is defined within <cxxDefine>. As well as the name and location of the #define, you can provide an example of its use with the <example>child element.
