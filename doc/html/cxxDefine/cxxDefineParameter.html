cxxDefineParameter

When a #define is used as preprocessor macro, the <cxxDefineParameter> element describes a named macro parameter.

Description

Macros can be created using the preprocessor #define directive. They can take arguments in a similar way to C++ functions. However macro arguments are untyped. The <cxxDefineParameter> element describes a single argument, or parameter, in a marco argument list.

Example

<cxxDefineDefinition>
    <cxxDefinePrototype>#define MergeModes(a, b)</cxxDefinePrototype>
    <cxxDefineNameLookup>MergeModes</cxxDefineNameLookup>
    <cxxDefineParameters>
        <cxxDefineParameter>
            <cxxDefineParameterDeclarationName>a</cxxDefineParameterDeclarationName>
            <apiDefNote/>
        </cxxDefineParameter>
        <cxxDefineParameter>
            <cxxDefineParameterDeclarationName>b</cxxDefineParameterDeclarationName>
            <apiDefNote/>
        </cxxDefineParameter>
    </cxxDefineParameters>
    <cxxDefineAPIItemLocation>
        <cxxDefineDeclarationFile name="filePath" value="C:/EPOC/master/sf/os/commsfw/serialserver/c32serialserver/SCOMM/CS_PORT.CPP"/>
        <cxxDefineDeclarationFileLine name="lineNumber" value="126"/>
    </cxxDefineAPIItemLocation>
</cxxDefineDefinition>
