cxxClassNestedDetail

The <cxxClassNestedDetail> element provides details about the nested members of a C++ class.

Description

In the <cxxClassNestedDetail> element, the <cxxClassNestedClass>, <cxxClassNestedStruct> and <cxxClassNestedUnion> elements are provided to indicate the members that are nested by a C++ class. <cxxClassNestedDetail> must have at least one of these child elements and each one can occur multiple times.

Example

<cxxClassNested>
    <cxxClassNestedDetail>
        <cxxClassNestedUnion href="GUID-D2E1A4E9-C8AB-34C3-827D-FB83D065899D.xml">CommsFW::TCFMessageBase::TCFMessageData</cxxClassNestedUnion>
        <cxxClassNestedClass href="GUID-FAF214B2-1CCA-385A-882F-AEB031273917.xml">RMobileSmsMessaging::TMobileSmsSendAttributesV1</cxxClassNestedClass>
        <cxxClassNestedStruct href="GUID-98F21850-A483-3C81-A2F2-72032E0F9448.xml">RFax::TProgress</cxxClassNestedStruct>            
    </cxxClassNestedDetail>
</cxxClassNested>
