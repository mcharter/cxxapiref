cxxClassBaseClass

Specifies the base class from which the current class derives.

Description

Inheritance is a relationship between classes. The derived class inherits from the base class. The derived class must indicate from which base class it inherits. In doing so it gains the functionality of the base class.

Example

<cxxClassDerivation>
    <cxxClassDerivationAccessSpecifier value="public"/>
    <cxxClassBaseClass href="class_base">Base</cxxClassBaseClass>
</cxxClassDerivation>
