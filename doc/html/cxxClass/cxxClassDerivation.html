cxxClassDerivation

Used to indicate that the current class derives from a base class. Child elements specify the name of the base class and whether public, protected, or private inheritance is used.

Description

The <cxxClassDerivation> element is used within a <cxxClassDerivations> element to specify a single class derivation. <cxxClassDerivation> must contain two child elements:
  • <cxxClassDerivationAccessSpecifier>
  • one of <cxxClassBaseClass>, <cxxClassBaseStruct> or <cxxClassBaseUnion>.

Example

<cxxClassDerivations>
	<cxxClassDerivation>
		<cxxClassDerivationAccessSpecifier value="public"/>
		<cxxClassBaseClass href="class_base">Base</cxxClassBaseClass>
	</cxxClassDerivation>
	...
</cxxClassDerivations>
