cxxClassDeclarationFileLineEnd

The <cxxClassDeclarationFileLineEnd> element specifies the end of the class declaration.

Description

The <cxxClassDeclarationFileLineEnd> element is defined within <cxxClassAPIItemLocation>. It contains the line number that the class declaration ends on. The element is empty but for the required attributes name and value. The attribute name must be set to "lineNumber". The attribute value is set to the line number the class declaration ends on.

Example

<cxxClassAPIItemLocation>
    <cxxClassDeclarationFile name="filePath" value="D:/include/headerfile.h"/>
    <cxxClassDeclarationFileLineStart name="lineNumber" value="765"/>
    <cxxClassDeclarationFileLineEnd name="lineNumber" value="787"/>
</cxxClassAPIItemLocation>
