cxxClassTemplateParameterType

The <cxxClassTemplateParameterType> allows you to provide information about the purpose of a template parameter.

Description

When instantiating a class template a developer needs to know the role of the template parameter and any constraints on the type used. The <cxxClassTemplateParameterType> element is defined within the <cxxClassTemplateParameter> element. Use it together with the <apiDefNote> sibling element to document how a class template should be used.

Example

<cxxClassTemplateParameters>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename T</cxxClassTemplateParameterType>
    <apiDefNote>The type of the first function object. </apiDefNote>
  </cxxClassTemplateParameter>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename U</cxxClassTemplateParameterType>
    <apiDefNote>The type of the second function object. </apiDefNote>
  </cxxClassTemplateParameter>
</cxxClassTemplateParameters>
