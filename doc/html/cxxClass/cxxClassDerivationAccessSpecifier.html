cxxClassDerivationAccessSpecifier

Indicates the access specifier used when a class inherits from another class.

Description

The <cxxClassDerivationAccessSpecifier> element is contained within a <cxxClassDerivation> element. The mandatory value attribute is set to “public”, “protected” or “private” to describe the access the class has to the members of the base class.

Use the <cxxClassDerivationAccessSpecifier> element together with the sibling element <cxxClassBaseClass>. This specifies the base class. Alternatively you can use <cxxClassBaseStruct> or <cxxClassBaseUnion> if you derive from a struct or union instead of from a class.

Example

<cxxClassDerivation>
	<cxxClassDerivationAccessSpecifier value="public"/>
	<cxxClassBaseClass href="class_base">Base</cxxClassBaseClass>
</cxxClassDerivation>
