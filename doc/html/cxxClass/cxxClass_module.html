cxxClass module

Defines a top-level container for all the elements necessary to describe an C++ API class.

Description

The <cxxClass> topic type contains elements necessary to show the name, short description, syntax and usage of a class.

<cxxClass> also contains all the elements to describe the member variables and functions. To do this insert cxxFunction, cxxDefine, cxxVariable, cxxEnumeration or cxxTypedef elements into the cxxClass module.

The cxxClass topic contains the following elements:

  • cxxClass
    Provides the name of a C++ class, all the elements necessary to describe the members of the class, and the members inherited from the immediate base class.
  • cxxClassAbstract
    Can optionally be used to mark a class as abstract.
  • cxxClassAccessSpecifier
    Describes the level of access allowed to the member variables or functions of the class.
  • cxxClassAPIItemLocation
    Contains child elements that specify the location of the declaration and definition of the members of a class.
  • cxxClassBaseClass
    Specifies the base class from which the current class derives.
  • cxxClassBaseStruct
    Specifies the struct from which the current class derives.
  • cxxClassBaseUnion
    Indicates inheritance from a union in a class derivation.
  • cxxClassDeclarationFile
    Specifies the location of the header file where the class is declared.
  • cxxClassDeclarationFileLineEnd
    The <cxxClassDeclarationFileLineEnd> element specifies the end of the class declaration.
  • cxxClassDeclarationFileLineStart
    The <cxxClassDeclarationFileLineStart> element specifies the beginning of the class declaration.
  • cxxClassDefinition
    Used to introduce a new class definition. Child elements describe the details of the class.
  • cxxClassDerivation
    Used to indicate that the current class derives from a base class. Child elements specify the name of the base class and whether public, protected, or private inheritance is used.
  • cxxClassDerivationAccessSpecifier
    Indicates the access specifier used when a class inherits from another class.
  • cxxClassDerivations
    Specifies the base class or base classes from which the class derives. Use a separate child <cxxClassDerivation> element to represent each base class.
  • cxxClassDerivationVirtual
    Use <cxxClassDerivationVirtual> element to indicate you are using virtual inheritance.
  • cxxClassDetail
    The <cxxClassDetail> element provides detailed information about the behavior and implementation of a C++ class.
  • cxxClassEnumerationInherited
    The <cxxClassEnumerationInherited> element within <cxxClassInheritsDetail> provides the name of an enumeration inherited by a class.
  • cxxClassEnumeratorInherited
    An enumerator is a named integer constant introduced by an enumeration type. The <cxxClassEnumeratorInherited> element within <cxxClassInheritsDetail> provides the name of an enumerator inherited by a class.
  • cxxClassFunctionInherited
    The <cxxClassFunctionInherited> element within <cxxClassInheritsDetail> provides the name of a function inherited by a class.
  • cxxClassInherits
    The <cxxClassInherits> element is a child of the <cxxClass> element. It contains <cxxClassInheritsDetail> which gives details of inherited members of the class.
  • cxxClassInheritsDetail
    The <cxxClassInheritsDetail> element provides details about the inherited members of a C++ class.
  • cxxClassNested
    Use <cxxClassNested> element within <cxxClass> to describe a nested class. The child element <cxxClassNestedDetail> gives details the members of the nested class.
  • cxxClassNestedClass
    The <cxxClassNestedClass> element within <cxxClassNestedDetail> provides the name of a nested class.
  • cxxClassNestedDetail
    The <cxxClassNestedDetail> element provides details about the nested members of a C++ class.
  • cxxClassNestedStruct
    The <cxxClassNestedStruct> element within <cxxClassNestedDetail> provides the name of a nested struct.
  • cxxClassNestedUnion
    The <cxxClassNestedUnion> element within <cxxClassNestedDetail> provides the name of a nested union.
  • cxxClassTemplateParameter
    The <cxxClassTemplateParameter> element describes one of the template types used by a C++ class template.
  • cxxClassTemplateParameters
    Use <cxxClassTemplateParameters> to indicate the template type or types used by a C++ class template. <cxxClassTemplateParameters> is declared within <cxxClassDefinition>.
  • cxxClassTemplateParameterType
    The <cxxClassTemplateParameterType> allows you to provide information about the purpose of a template parameter.
  • cxxClassTypedefInherited
    As the author of a derived class it is useful to be aware of any typedefs inherited from the base class. The <cxxClassTypedefInherited> element links to an typedef that has been declared in the scope of a base class.
  • cxxClassVariableInherited
    Provides the name of a variable inherited by a class.
