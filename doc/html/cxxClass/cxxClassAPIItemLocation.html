cxxClassAPIItemLocation

Contains child elements that specify the location of the declaration and definition of the members of a class.

Description

The <cxxClassAPIItemLocation> element must contain the location of the API declaration using the elements <cxxClassDeclarationFile> and <cxxClassDeclarationFileLine>. The location of the API definition is optional.

Example

<cxxClassAPIItemLocation>
				<cxxClassDeclarationFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxClassDeclarationFileLine name="lineNumber" value="775"/>
				<cxxClassDefinitionFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxClassDefinitionFileLineStart name="lineNumber" value="765"/>
				<cxxClassDefinitionFileLineEnd name="lineNumber" value="787"/>
			</cxxClassAPIItemLocation>
