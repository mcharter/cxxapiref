cxxClassTemplateParameters

Use <cxxClassTemplateParameters> to indicate the template type or types used by a C++ class template. <cxxClassTemplateParameters> is declared within <cxxClassDefinition>.

Description

The <cxxClassTemplateParameters> element contains one or more <cxxClassTemplateParameter> elements. Each <cxxClassTemplateParameter> element describes a single type on which the class is parameterized.

Example

<cxxClassTemplateParameters>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename T</cxxClassTemplateParameterType>
    <apiDefNote>The type of the first function object. </apiDefNote>
  </cxxClassTemplateParameter>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename U</cxxClassTemplateParameterType>
    <apiDefNote>The type of the second function object. </apiDefNote>
  </cxxClassTemplateParameter>
</cxxClassTemplateParameters>
