cxxClassNested

Use <cxxClassNested> element within <cxxClass> to describe a nested class. The child element <cxxClassNestedDetail> gives details the members of the nested class.

Description

<cxxClassNested> contains one mandatory <cxxClassNestedDetail> element and no attributes.

Example

<cxxClassNested>
    <cxxClassNestedDetail>
        <cxxClassNestedUnion href="GUID-D2E1A4E9-C8AB-34C3-827D-FB83D065899D.xml">CommsFW::TCFMessageBase::TCFMessageData</cxxClassNestedUnion>
        <cxxClassNestedClass href="GUID-FAF214B2-1CCA-385A-882F-AEB031273917.xml">RMobileSmsMessaging::TMobileSmsSendAttributesV1</cxxClassNestedClass>
        <cxxClassNestedStruct href="GUID-98F21850-A483-3C81-A2F2-72032E0F9448.xml">RFax::TProgress</cxxClassNestedStruct>            
    </cxxClassNestedDetail>
</cxxClassNested>
