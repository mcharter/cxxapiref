cxxClassBaseStruct

Specifies the struct from which the current class derives.

Description

Use the <cxxClassBaseStruct> for the rare case of inheriting from a struct. It is more common to inherit from a class.

Example

<cxxClassDerivation>
    <cxxClassDerivationAccessSpecifier value="public"/>
    <cxxClassBaseStruct href="struct_base">Base</cxxClassBaseStruct>
</cxxClassDerivation>
