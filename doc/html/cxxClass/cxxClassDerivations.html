cxxClassDerivations

Specifies the base class or base classes from which the class derives. Use a separate child <cxxClassDerivation> element to represent each base class.

Description

Use <cxxClassDerivations> within a <cxxClassDefinition> element to specify any class derivations (i.e. base classes) for a class. In the case of single inheritance from a class there will be a single <cxxClassDerivation> child element. In the case of multiple inheritance the list of class derivations will have multiple members. <cxxClassDerivations> must contain at least one child element. This will be one of the following:
  • <cxxClassDerivation>
  • <cxxStructDerivation>
  • <cxxUnionDerivation>

Example

<cxxClassDefinition>
	<cxxClassDerivations>
		<cxxClassDerivation>
			...
		</cxxClassDerivation>
		...
	</cxxClassDerivations>
	<cxxClassAPIItemLocation>
		...
	</cxxClassAPIItemLocation>
<cxxClassDefinition>
