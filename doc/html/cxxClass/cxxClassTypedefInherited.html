cxxClassTypedefInherited

As the author of a derived class it is useful to be aware of any typedefs inherited from the base class. The <cxxClassTypedefInherited> element links to an typedef that has been declared in the scope of a base class.

Description

A typedef is a new name for an existing type. A typedef defined inside the scope of a base class can be used also inside a derived class.

Example

<cxxClassInherits>
    <cxxClassInheritsDetail>
        <cxxClassTypedefInherited href="aclass.xml#atypedef">AClass::ATypedef</cxxClassTypedefInherited>
    </cxxClassInheritsDetail>
</cxxClassInherits>
