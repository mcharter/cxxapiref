cxxClassAbstract

Can optionally be used to mark a class as abstract.

Description

The <cxxClassAbstract> element resides within the <cxxClassDefinition> element. The <cxxClassAbstract> element has no children but has the attributes name and value. These attributes are both set to "abstract".
