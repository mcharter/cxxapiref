cxxClass

Provides the name of a C++ class, all the elements necessary to describe the members of the class, and the members inherited from the immediate base class.

Description

<cxxClass> requires an ID attribute.

<cxxClass> requires the child elements <apiName> and <cxxClassDetail>.
