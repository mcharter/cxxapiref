cxxClassVariableInherited

Provides the name of a variable inherited by a class.

Description

The <cxxClassVariableInherited> element is a child of <cxxClassInheritsDetail>. The text in this element is the fully qualified name of the variable that has been inherited. There are no mandatory attributes.

Example

<cxxClassInherits>
    <cxxClassInheritsDetail>
        <cxxClassVariableInherited href="file.xml">CActive::iStatus</cxxClassVariableInherited>
    </cxxClassInheritsDetail>
</cxxClassInherits>
