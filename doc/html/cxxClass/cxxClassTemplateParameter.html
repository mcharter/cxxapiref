cxxClassTemplateParameter

The <cxxClassTemplateParameter> element describes one of the template types used by a C++ class template.

Description

The <cxxClassTemplateParameters> element contains one or more <cxxClassTemplateParameter> child elements. <cxxClassTemplateParameter> requires the child element <cxxClassTemplateParameterType> to indicate the type of parameter. <cxxClassTemplateParameter> may also contain the child element <apiDefNote> to describe the parameter further.

Example

<cxxClassTemplateParameters>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename T</cxxClassTemplateParameterType>
    <apiDefNote>The type of the first function object. </apiDefNote>
  </cxxClassTemplateParameter>
  <cxxClassTemplateParameter>
    <cxxClassTemplateParameterType>typename U</cxxClassTemplateParameterType>
    <apiDefNote>The type of the second function object. </apiDefNote>
  </cxxClassTemplateParameter>
</cxxClassTemplateParameters>
