cxxFunctionConst

Used to describe a function with a const cv-qualifier.

Description

The <cxxFunctionConst> element is present when the function has a const cv-qualifier . The element has two mandatory attributes, name and value. If present, these attributes must be set to the value “const”.

See ISO/IEC 14882:2003(E) 7.1.5.1 The cv-qualifiers [dcl.type.cv]

Example

Given this code:

struct X {
    void g() const;
};

Then function g would have the cxxFunctionConst element:

<cxxFunctionConst />

If the mandatory attributes are present they must be set thus:

<cxxFunctionConst name="const" value="const" />
