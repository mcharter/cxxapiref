cxxFunctionParameterDeclarationName

Describes the name a function parameter, if the name is present when the function is declared.

Description

The <cxxFunctionParameterDeclarationName> element specifies the name of one of the parameter used in the declaration of the function. This will normally either be absent or be the same as the corresponding parameter name used in the function definition.

See ISO/IEC 14882:2003(E) 8.3.5 Functions [dcl.fct]
