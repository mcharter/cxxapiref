cxxFunctionReturnDesc

Describes the return value of a function.

Description

The <cxxFunctionReturnDesc> element is optional within a function definition. It provides a textual description of a function return value.

Example

<cxxFunctionDefinition>
	<cxxFunctionReturnDesc>The index in the buffer or -1 if not found or i is &gt;= the buffer length. </cxxFunctionReturnDesc>     
</cxxFunctionDefinition>
