cxxFunctionDefinition

Describes all of the details of a C++ function declaration.

Description

The <cxxFunctionDefinition> element contains child elements that together fully describe a C++ function declaration. As well as the function signature and return type, there are child elements to indicate whether the function is virtual, whether it is parameterized, and any modifiers applied to the function.

The content model reflects the complexity of C++ function declarations.
