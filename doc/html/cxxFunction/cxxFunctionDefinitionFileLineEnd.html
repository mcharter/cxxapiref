cxxFunctionDefinitionFileLineEnd

Identifies the line number in file that contains the end of the definition of the function.

Description

The <cxxFunctionDefinitionFileLineEnd> element identifies the line number in file that contains the end of the definition of the function. There are two attributes. The name attribute always has the value "lineNumber", and the value attribute has the line number where the function definition ends.

Note: ISO/IEC 14882:2003(E) 16.4 Line control [cpp.line] means that line numbers start from 1.

Example

<cxxFunctionDefinitionFileLineEnd name="lineNumber" value="7"/>
