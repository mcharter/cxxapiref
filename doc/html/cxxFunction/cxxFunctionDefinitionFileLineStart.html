cxxFunctionDefinitionFileLineStart

Identifies the line number in file that contains the start of the definition of the function.

Description

The <cxxFunctionDefinitionFileLineStart> element identifies the line number in file that contains the start of the definition of the function. There are two attributes. The name attribute always has the value "lineNumber", and the value attribute has the line number where the function definition begins.

Note: ISO/IEC 14882:2003(E) 16.4 Line control [cpp.line] means that line numbers start from 1.

Example

<cxxFunctionDefinitionFileLineStart name="lineNumber" value="4"/>
