cxxFunctionReimplemented

Identifies functions that have been re-implemented by a derived class.

Description

The <cxxFunctionReimplemented> element identifies and links to functions that have been overridden in a derived class.

Example

Given this code:

class Base
{
public:
    virtual void f();
};

class Derived : public Base
{
public:
    void f();
};

Then function Derived::f would have the cxxFunctionReimplemented element thus:

<cxxFunctionReimplemented href="class_base.xml#class_base_1a1b981706372d75125af29b377d33ba66">Base::f()</cxxFunctionReimplemented>
