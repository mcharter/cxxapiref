cxxFunctionExplicit

Marks a function (that must always be a constructor) as qualified with the C++ keyword explicit.

Description

The<cxxFunctionExplicit> element marks a function as explicit. The function must be a constructor. The element has two mandatory attributes: name and value .

See ISO/IEC 14882:2003(E) 7.1.2 Function specifiers [dcl.fct.spec]

The explicit specifier shall be used only in declarations of constructors within a class declaration; see ISO/IEC 14882:2003(E) 12.3.1.
