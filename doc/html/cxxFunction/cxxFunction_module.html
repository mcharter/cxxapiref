cxxFunction module

The cxxFunction module describes C++ functions, including member functions.

Description

The <cxxFunction> topic type contains elements to show the declaration, location, and usage of a function.

Common elements used to describe a function are:
  • cxxFunctionAccessSpecifier
  • cxxFunctionDeclaredType
  • cxxFunctionScopedName
  • cxxFunctionPrototype
  • cxxFunctionNameLookup
  • cxxFunctionAPIItemLocation

The cxxFunction topic contains the following elements:

  • cxxFunction
    Contains all the elements to describe a C++ function or member function.
  • cxxFunctionAccessSpecifier
    Describes the access specifier for the function ("public", "protected" or "private").
  • cxxFunctionAPIItemLocation
    Describes the location of a function in the C++ source code.
  • cxxFunctionConst
    Used to describe a function with a const cv-qualifier.
  • cxxFunctionConstructor
    Identifies a function as a constructor.
  • cxxFunctionDeclarationFile
    Identifies the path to the file that contains the declaration of the function.
  • cxxFunctionDeclarationFileLine
    Identifies the line number in file that contains the declaration of the function.
  • cxxFunctionDeclaredType
    Identifies the return type of the function.
  • cxxFunctionDefinition
    Describes all of the details of a C++ function declaration.
  • cxxFunctionDefinitionFile
    Identifies the path to the file that contains the definition of the function.
  • cxxFunctionDefinitionFileLineEnd
    Identifies the line number in file that contains the end of the definition of the function.
  • cxxFunctionDefinitionFileLineStart
    Identifies the line number in file that contains the start of the definition of the function.
  • cxxFunctionDestructor
    Identifies a function as a destructor.
  • cxxFunctionDetail
    Contains the definition of a C++ function and optionally further information about the function.
  • cxxFunctionExceptionSpecification
    Contains the exception specification of a C++ function.
  • cxxFunctionExplicit
    Marks a function (that must always be a constructor) as qualified with the C++ keyword explicit.
  • cxxFunctionInline
    Identifies the function as marked with the C++ keyword inline.
  • cxxFunctionNameLookup
    Provides a unique identifier for the function.
  • cxxFunctionParameter
    Contains child elements to describe the name and type of a function parameter.
  • cxxFunctionParameterDeclarationName
    Describes the name a function parameter, if the name is present when the function is declared.
  • cxxFunctionParameterDeclaredType
    Describes the type of a function parameter.
  • cxxFunctionParameterDefaultValue
    Describes any default value that the function parameter has.
  • cxxFunctionParameterDefinitionName
    Describes the name of the parameter used in the definition of the function.
  • cxxFunctionParameters
    Contains child elements describing the type and name of each of the function parameters.
  • cxxFunctionPrototype
    Provides a "visual" representation of the function declaration.
  • cxxFunctionPureVirtual
    Identifies a function as pure virtual.
  • cxxFunctionReimplemented
    Identifies functions that have been re-implemented by a derived class.
  • cxxFunctionReturnDesc
    Describes the return value of a function.
  • cxxFunctionScopedName
    Identifies the fully qualified scope of the function, if it is not at the global scope. The scope is the enclosing namespace/class/struct/union.
  • cxxFunctionStorageClassSpecifierExtern
    Identifies functions that have external linkage.
  • cxxFunctionStorageClassSpecifierMutable
    Describes functions that have a mutable storage-class-specifier.
  • cxxFunctionStorageClassSpecifierStatic
    Describes a function with static linkage.
  • cxxFunctionTemplateParameter
    Contains the type of the template parameter and any documentation associated with that parameter.
  • cxxFunctionTemplateParameters
    Container for elements to describe each of the template parameters of a function template.
  • cxxFunctionTemplateParameterType
    Describes the type of a template parameter of a function template.
  • cxxFunctionVirtual
    Identifies a function as virtual.
  • cxxFunctionVolatile
    Used to describe a function with a volatile cv-qualifier.
