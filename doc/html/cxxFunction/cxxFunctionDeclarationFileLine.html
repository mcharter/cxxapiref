cxxFunctionDeclarationFileLine

Identifies the line number in file that contains the declaration of the function.

Description

The <cxxFunctionDeclarationFileLine> element identifies the line number in the file that contains the declaration of the function. There are two attributes. The name attribute always has the value "lineNumber", and the value attribute has the line number where the declaration is made.

Note: ISO/IEC 14882:2003(E) 16.4 Line control [cpp.line] means that line numbers start from 1.

Example

<cxxFunctionDeclarationFileLine name="lineNumber" value="5"/>
