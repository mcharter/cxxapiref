cxxFunctionPureVirtual

Identifies a function as pure virtual.

Description

The <The cxxFunctionPureVirtual> element has two mandatory attributes, name and value . Both attributes default to "pure virtual".

A function that uses the pure-specifier is a pure virtual function. See: ISO/IEC 14882:2003(E) 10.4 Abstract classes [class.abstract]
