cxxFunctionAccessSpecifier

Describes the access specifier for the function ("public", "protected" or "private").

Description

The <cxxFunctionAccessSpecifier> describes the access specifier according to SO/IEC 14882:2003(E) 11 Member access control [class.access].

The element has a name attribute that must have a value of “public”, “protected”, or “private”. It defaults to "public".
