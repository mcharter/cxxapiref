cxxFunctionDetail

Contains the definition of a C++ function and optionally further information about the function.

Description

The <cxxFunctionDetail> element contains a mandatory child element<cxxFunctionDefinition> and an optional child element<apiDesc> that can be used to document the operation of the function.
