cxxFunctionTemplateParameter

Contains the type of the template parameter and any documentation associated with that parameter.

Description

The <cxxFunctionTemplateParameter> element had child elements <cxxFunctionTemplateParameterType> and <apiDefNote>. These child elements can be used to name template parameter of a function template and to describe how the parameter should be used.

See ISO/IEC 14882:2003(E) 14.1 Template parameters [temp.param]
