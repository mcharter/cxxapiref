cxxFunctionDeclarationFile

Identifies the path to the file that contains the declaration of the function.

Description

The <cxxFunctionDeclarationFile> element identifies the path to the file that contains the declaration of the function. There are two attributes. The name attribute always has the value "filePath" and the value attribute has the pathname of the file where the declaration is made.

Example

<cxxFunctionDeclarationFile name="filePath" value="inc/a.h"/>
