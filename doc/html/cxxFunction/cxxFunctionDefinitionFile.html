cxxFunctionDefinitionFile

Identifies the path to the file that contains the definition of the function.

Description

The <cxxFunctionDefinitionFile> element identifies the path to the file that contains the definition of the function. There are two attributes. The name attribute always has the value "filePath" and the value attribute has the pathname of the file where the definition is made.

Example

<cxxFunctionDefinitionFile name="filePath" value="inc/a.h"/>
