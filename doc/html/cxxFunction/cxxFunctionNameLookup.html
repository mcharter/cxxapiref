cxxFunctionNameLookup

Provides a unique identifier for the function.

Description

The <cxxFunctionNameLookup> element provides a way of uniquely identifying a function in a body of DITA XML that describes a body of C++ source code.

The free format text in this element's PCDATA should:
  • be unique within a set of DITA XML files.
  • be human readable.
  • follow the lookup techniques use in the C++ standard (ISO/IEC 14882:2003(E)).
  • use a reversible encoding.
