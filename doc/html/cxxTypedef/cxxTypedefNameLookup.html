cxxTypedefNameLookup

Provides a unique identifier for the typedef.

Description

This element provides a way of uniquely identifying a typedef in a body of DITA XML that describes a body of C++ source code.

The PCDATA contained within the <cxxTypedefNameLookup> element is free format. The PCDATA text should:
  • be unique within a set of DITA XML files.
  • be human readable.
  • follow the lookup techniques use in the C++ standard (ISO/IEC 14882:2003(E)).
  • use a reversible encoding.
