cxxTypedefAccessSpecifier

Indicates whether a typedef introduced inside a class definition is treated as public, protected, or private.

Description

When declared inside a class, typedefs respect the access modifiers. For example a typedef in the private section of a class will not be visible outside the class.

The <cxxTypedefAccessSpecifier> element does not have any children. It uses the attribute value to describe the level of access to the typedef. valuecan be set to "public", "protected" or "private"; the default is "public".

Example

<cxxTypedefAccessSpecifier value="public"/>
