cxxTypedefDeclarationFileLine

Specifies the line at which the typedef is introduced.

Description

The <cxxTypedefDeclarationFileLine> element is defined within <cxxTypedefAPIItemLocation>. <cxxTypedefDeclarationFileLine> has no content apart from the mandatory attributes name and value. Name is always “lineNumber”and valueis the location of the typedef within the source file.

Example

<cxxTypedefAPIItemLocation>
				<cxxTypedefDeclarationFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxTypedefDeclarationFileLine name="lineNumber" value="775"/>
				<cxxTypedefDefinitionFile name="filePath" value="D:/src/include/headerfile.h"/>
			</cxxTypedefAPIItemLocation>
