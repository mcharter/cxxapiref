cxxTypedefDeclaredType

Used to specify the existing type that is aliased by means of the typedef.

Description

The optional <cxxTypedefDeclaredType> element contains a CDATA field and an <apiRelation> element.

The CDATA field may be used to show any C++ keywords associated with the typedef (e.g. const or static). The <apiRelation> element provides a links to the documentation of the existing type referred to by the typedef.

Example

<cxxTypedefDeclaredType>const <apiRelation href="GUID-FC2F2B93-3D18-3BCC-9FD6-6BC6B240B667.xml">TUint</apiRelation></cxxTypedefDeclaredType>
