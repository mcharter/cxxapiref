cxxTypedef module

Defines a top-level container for all the elements necessary to describe an C++ API typedef.

Description

The <cxxTypedef> topic type contains elements necessary to show the name of the typedef, the typedef syntax, and the typedef use description. This topic type module also contains all the elements necessary to describe the component variables and functions.

The cxxTypedef topic contains the following elements:

  • cxxTypedef
    Describes a C++ typedef.
  • cxxTypedefAccessSpecifier
    Indicates whether a typedef introduced inside a class definition is treated as public, protected, or private.
  • cxxTypedefAPIItemLocation
    Contains child elements specifying the location a typedef in the source code..
  • cxxTypedefDeclarationFile
    Specifies the source code file containing the typedef.
  • cxxTypedefDeclarationFileLine
    Specifies the line at which the typedef is introduced.
  • cxxTypedefDeclaredType
    Used to specify the existing type that is aliased by means of the typedef.
  • cxxTypedefDefinition
    Contains child elements providing details of an alias introduced by the C++ keyword typedef.
  • cxxTypedefDetail
    The <cxxTypedefDetail> element can be used to provide further information about a C++ typedef.
  • cxxTypedefNameLookup
    Provides a unique identifier for the typedef.
  • cxxTypedefPrototype
    Provides a "visual" representation of the typedef declaration.
  • cxxTypedefReimplemented
    Identifies typedefs that have been re-implemented.
  • cxxTypedefScopedName
    Identifies the enclosing namespace/class/struct/union within which the typedef is introduced.
