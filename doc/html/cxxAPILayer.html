C++ topic types

The C++ API specialization provides a set of specific topic types that can describe a C++ application programming interface (API) library. The C++ language-specific API topic types are specialized from the generic API reference topic types. This specialization permits the sharing of the basic semantics and processing.

Description

The C++ modules describe the C++ topic types and their contained elements. For example, the cxxClass module describes all the elements necessary to describe a C++ class.

  • cxxapi-domain module
    Provides the description of the elements that refer to the API items within C++ API topics or other DITA topic types, such as concept or task.
  • cxxAPIMap module
    Provides the relationships among a set of C++ API topic types, and can be used to create a Table of Contents (TOC), to aggregate C++ API topics into a PDF document, or to create links between topics in output.
  • cxxClass module
    Defines a top-level container for all the elements necessary to describe an C++ API class.
  • cxxDefine module
    Defines a top-level container for all the elements necessary to describe a preprocessor definition.
  • cxxEnumeration module
    Defines a top-level container for all the elements necessary to describe an C++ enumeration.
  • cxxFile module
    The cxxFile module defines a top-level container for all the elements necessary to describe an API declared at the global scope.
  • cxxFunction module
    The cxxFunction module describes C++ functions, including member functions.
  • cxxNamespace module
    Defines a top-level container for all the elements necessary to describe a C++ namespace.
  • cxxStruct module
    Defines a top-level container for all the elements to describe a C++ struct.
  • cxxTypedef module
    Defines a top-level container for all the elements necessary to describe an C++ API typedef.
  • cxxUnion module
    Defines a top-level container for all the elements necessary to describe a C++ union.
  • cxxVariable module
    Defines a top-level container for all the elements to describe a member variable.
  • cxxFriend module
    Defines a top-level container for all the elements necessary to describe an C++ API friend.
