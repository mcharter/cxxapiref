cxxVariableDeclarationFileLine

Specifies the line on which a member variable is declared.

Description

The <cxxVariableDeclarationFileLine> element is defined within <cxxVariableAPIItemLocation>. The element has no children. It has the mandatory attributes name and value. The name attribute is always "lineNumber" and valueis the line number containing the variable declaration.

Example

<cxxVariableAPIItemLocation>
    <cxxVariableDeclarationFile name="filePath" value="C:/src/include/e32base.h"/>
    <cxxVariableDeclarationFileLine name="lineNumber" value="1664"/>
</cxxVariableAPIItemLocation>
