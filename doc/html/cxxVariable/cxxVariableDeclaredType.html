cxxVariableDeclaredType

Specifies the data type of a member variable.

Description

The <cxxVariableDeclaredType> element is optional. It contains a CDATA field and a <apiRelation> element. The CDATA field may be used to show any C++ keywords associated with the declaration e.g. const or static. The <apiRelation> element provides a link to the documentation of the data type of the variable.

Example

<cxxVariableDeclaredType>const <apiRelation href="GUID-FC2F2B93-3D18-3BCC-9FD6-6BC6B240B667.xml">TUint</apiRelation></cxxVariableDeclaredType>
