cxxVariableDetail

A container for child elements giving detailed information about a member variable.

Description

The <cxxVariableDetail> element is defined within <cxxVariable>. In <cxxVariableDetail> the <example> and <section> elements are provided to provide further information about member variable.

Contained by

cxxVariable

Contains

apiDesc, apiImpl, cxxVariableDefinition example, section,

Content Model

(cxxVariableDefinition then apiDesc, then (example, or section, or apiImpl, (any number) ))
