cxxVariableScopedName

Indicates the name of the immediately enclosing scope where a variable is declared.

Description

The value of the <cxxVariableScopedName> element indicates the namespace/class/struct/union within which a variable is declared. The element has no compulsory attributes and no child elements.

Example

<cxxVariableScopedName>CServer2</cxxVariableScopedName>
