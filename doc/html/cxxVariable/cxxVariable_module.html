cxxVariable module

Defines a top-level container for all the elements to describe a member variable.

Description

Common elements used to describe a member variable are:

  • cxxVariableAccessSpecifier
  • cxxVariableDeclaredType
  • cxxVariableScopedName
  • cxxVariablePrototype
  • cxxVariableNameLookup
  • cxxVariableAPIItemLocation

The cxxVariable topic contains the following elements:

  • cxxVariable
    Contains child elements to describe a C++ member variable.
  • cxxVariableAccessSpecifier
    Specifies the access modifier applied to a variable.
  • cxxVariableAPIItemLocation
    Contains child elements to specify the location of a member variable.
  • cxxVariableConst
    The <cxxVariableConst> element is optional. It is used to indicate that a member variable is qualified with the keyword const.
  • cxxVariableDeclarationFile
    Specifies the file containing the declaration of a member variable.
  • cxxVariableDeclarationFileLine
    Specifies the line on which a member variable is declared.
  • cxxVariableDeclaredType
    Specifies the data type of a member variable.
  • cxxVariableDefinition
    A container for child elements giving further information about a member variable. All elements within <cxxVariableDefinition> are optional.
  • cxxVariableDetail
    A container for child elements giving detailed information about a member variable.
  • cxxVariableNameLookup
    An optional element to specify the lookup name of a variable.
  • cxxVariablePrototype
    An optional element providing a representation of the variable declaration.
  • cxxVariableReimplemented
    Identifies a variable that has been re-implemented in a derived class.
  • cxxVariableScopedName
    Indicates the name of the immediately enclosing scope where a variable is declared.
  • cxxVariableStorageClassSpecifierExtern
    Indicates that a variable is marked as storage class extern. This is so that a variable can be used in a separate source file (or “translation unit”) to the one where it is originally defined.
  • cxxVariableStorageClassSpecifierMutable
    Marks a variable as qualified with the C++ keyword mutable.
  • cxxVariableStorageClassSpecifierStatic
    Marks a variable as qualified with the C++ keyword static.
  • cxxVariableVolatile
    Marks a variable as qualified with the C++ keyword volatile.
