cxxVariableNameLookup

An optional element to specify the lookup name of a variable.

Description

Thel <cxxVariableNameLookup> element has no child elements and does not require any attributes. The CDATA text inside <cxxVariableNameLookup> is the lookup name of a variable.

Example

<cxxVariableNameLookup>CActive::iStatus</cxxVariableNameLookup>
