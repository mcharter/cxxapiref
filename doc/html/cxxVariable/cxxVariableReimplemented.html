cxxVariableReimplemented

Identifies a variable that has been re-implemented in a derived class.

Description

The <cxxVariableReimplemented> element identifies and links to a variable that has been re-implemented. As an example, a base class sometimes contains a “spare” variable so that a derived class can add data without causing a derived class object to take up more memory than a base class object. This can be important for the purpose of binary compatability.

<cxxVariableReimplemented> has no mandatory attributes.

Example

<cxxVariableReimplemented href="class_c_active.xml#class_c_active_1a4bb76130b67b5430b3b736194df7edeb">CActive::iSpare</cxxVariableReimplemented>
